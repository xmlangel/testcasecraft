## 3. 주요 리포트 종류 및 기능

### 3.1. 테스트 계획 및 실행 결과 리포트
- **Test Plan Report**: 테스트 계획의 상세 내용을 문서화합니다.
- **Test Report**: 테스트 실행 결과를 종합하여 리포트를 생성합니다.
- **Test Report on Build**: 특정 빌드에 대한 테스트 실행 결과를 리포트로 생성합니다.

### 3.2. 측정 및 통계 리포트 (Metrics)
- **General Test Plan Metrics**: 테스트 계획의 전반적인 통계(테스트케이스 수, 실행률 등)를 제공합니다.
- **Results by Test Suite**: 테스트 스위트별로 테스트 결과를 집계하여 보여줍니다.
- **Results by Tester per Build**: 빌드별, 테스터별 테스트 수행 결과를 보여줍니다.
- **Charts**: 테스트 결과를 파이 차트 등 시각화된 형태로 보여줍니다.
- **Execution Timeline**: 시간의 흐름에 따른 테스트 실행 통계를 보여줍니다.

### 3.3. 상태 기반 리포트
- **Failed Test Cases**: 실패한 테스트케이스 목록을 보여줍니다.
- **Blocked Test Cases**: 블록된 테스트케이스 목록을 보여줍니다.
- **Not Run Test Cases**: 아직 실행되지 않은 테스트케이스 목록을 보여줍니다.
- **Never Run Test Cases**: 한 번도 실행된 적 없는 테스트케이스 목록을 보여줍니다.

### 3.4. 요구사항 및 커버리지 리포트
- **Requirements Coverage**: 요구사항이 테스트케이스에 의해 얼마나 커버되는지 보여줍니다.
