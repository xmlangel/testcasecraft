# 네비게이션 바 사용 매뉴얼

## 1. 개요

이 문서는 테스트케이스 관리 툴의 상단 네비게이션 바 사용 방법을 안내합니다. 네비게이션 바는 애플리케이션의 주요 섹션으로 빠르게 이동하고, 사용자 관련 기능(프로필, 로그아웃)에 접근하며, JIRA 연동 상태를 확인할 수 있는 핵심적인 요소입니다.

## 2. 네비게이션 바 구성 요소

네비게이션 바는 다음과 같은 주요 요소들로 구성됩니다.

### 2.1. 애플리케이션 제목

*   **표시**: "테스트케이스 관리툴"
*   **기능**: 현재 애플리케이션의 이름을 나타냅니다. 클릭 시 특별한 동작은 없습니다.

### 2.2. 주요 메뉴 버튼

애플리케이션의 주요 기능 영역으로 이동할 수 있는 버튼들입니다.

*   **대시보드**:
    *   **기능**: 프로젝트 및 조직 전반의 통계와 현황을 한눈에 볼 수 있는 대시보드 페이지로 이동합니다.
    *   **경로**: `/dashboard`
*   **조직 관리**:
    *   **기능**: 조직 생성, 수정, 삭제 및 조직 내 사용자 관리를 할 수 있는 페이지로 이동합니다.
    *   **경로**: `/organizations`
*   **사용자 관리**:
    *   **기능**: 시스템에 등록된 사용자 목록을 확인하고, 사용자 정보 수정 및 권한 관리를 할 수 있는 페이지로 이동합니다.
    *   **경로**: `/users`
*   **프로젝트 선택**:
    *   **기능**: 현재 활성화된 프로젝트를 변경하거나, 새로운 프로젝트를 생성/선택할 수 있는 페이지로 이동합니다.
    *   **경로**: `/projects`

### 2.3. JIRA 상태 인디케이터

*   **표시**: JIRA 연동 상태를 나타내는 아이콘 또는 텍스트 (예: JIRA 로고, 연결 상태 메시지)
*   **기능**: JIRA 서버와의 연결 상태를 시각적으로 보여줍니다. 연결 문제 발생 시 사용자에게 알림을 제공할 수 있습니다.

### 2.4. 사용자 메뉴

사용자 계정 관련 기능에 접근할 수 있는 메뉴입니다.

*   **표시**: 현재 로그인한 사용자의 이름 첫 글자를 표시하는 아바타 아이콘
*   **기능**: 클릭 시 드롭다운 메뉴가 나타나며, 다음 옵션들을 제공합니다.
    *   **프로필**:
        *   **기능**: 사용자 프로필 정보를 확인하고 수정할 수 있는 팝업 창을 엽니다.
    *   **로그아웃**:
        *   **기능**: 현재 세션을 종료하고 로그인 페이지로 이동합니다.

## 3. 네비게이션 바 사용 방법

1.  **페이지 이동**: 네비게이션 바 중앙에 위치한 "대시보드", "조직 관리", "사용자 관리", "프로젝트 선택" 버튼을 클릭하여 해당 기능 페이지로 이동합니다.
2.  **사용자 기능 접근**: 네비게이션 바 우측 상단의 사용자 아바타 아이콘을 클릭하여 드롭다운 메뉴를 엽니다.
    *   "프로필"을 클릭하여 사용자 정보를 관리합니다.
    *   "로그아웃"을 클릭하여 현재 세션을 종료합니다.
3.  **JIRA 상태 확인**: JIRA 상태 인디케이터를 통해 JIRA 서버와의 연결 상태를 주기적으로 확인합니다.

## 4. 반응형 동작

네비게이션 바는 다양한 화면 크기(데스크톱, 태블릿, 모바일)에 맞춰 자동으로 레이아웃이 조정됩니다. 화면 크기가 줄어들 경우, 일부 메뉴 항목이 숨겨지거나 다른 형태로 표시될 수 있습니다. (현재 구현에서는 데스크톱 레이아웃을 기준으로 설명합니다.)
