# 사용자 관리 페이지 사용 매뉴얼

## 1. 개요

이 문서는 테스트케이스 관리 시스템의 사용자 관리 페이지 사용 방법을 안내합니다. 사용자 관리 페이지는 시스템 관리자가 사용자 계정을 효율적으로 관리할 수 있도록 다양한 기능과 인터페이스를 제공합니다.

## 2. 주요 기능 및 구성 요소

### 2.1. 사용자 통계 요약

페이지 상단에는 시스템의 사용자 현황을 한눈에 파악할 수 있는 통계 카드가 표시됩니다.

*   **전체 사용자**: 시스템에 등록된 총 사용자 수
*   **활성 사용자**: 현재 활성화 상태인 사용자 수
*   **비활성 사용자**: 현재 비활성화 상태인 사용자 수
*   **최근 가입**: 최근에 가입한 사용자 수

### 2.2. 사용자 목록 툴바 (검색, 필터, 작업)

사용자 목록 상단에는 사용자를 검색하고 필터링하며, 특정 작업을 수행할 수 있는 툴바가 있습니다.

*   **검색 필드**: 사용자명, 이름, 이메일 등 키워드를 입력하여 사용자를 검색할 수 있습니다.
*   **역할 필터**: 드롭다운 메뉴를 통해 사용자의 역할(예: ADMIN, MANAGER, TESTER, USER)별로 목록을 필터링할 수 있습니다.
*   **상태 필터**: 드롭다운 메뉴를 통해 사용자의 활성/비활성 상태별로 목록을 필터링할 수 있습니다.
*   **새로고침 버튼**: 현재 사용자 목록을 새로고침하여 최신 상태를 반영합니다.
*   **데이터 내보내기 버튼**: 현재 조회된 사용자 목록 데이터를 파일로 내보낼 수 있습니다.
*   **초기화 버튼**: 모든 검색 및 필터 조건을 초기화하고 전체 사용자 목록을 다시 표시합니다.

### 2.3. 사용자 목록 테이블

시스템에 등록된 사용자들의 상세 정보가 테이블 형태로 표시됩니다.

*   **표시 항목**: 사용자명, 이름, 이메일, 역할, 상태, 가입일, 최종 로그인 정보가 각 열에 표시됩니다.
*   **정렬 기능**: 사용자명, 이름, 가입일, 최종 로그인 열의 헤더를 클릭하여 오름차순 또는 내림차순으로 정렬할 수 있습니다.
*   **역할 및 상태 표시**: 역할은 아이콘과 색상 칩으로, 상태는 활성/비활성에 따라 색상 칩으로 시각적으로 구분됩니다.
*   **작업 열**: 각 사용자 행의 마지막 열에는 해당 사용자에 대한 작업을 수행할 수 있는 아이콘 버튼이 있습니다.

### 2.4. 페이지네이션

사용자 목록이 많을 경우, 페이지네이션 컨트롤을 통해 목록을 탐색할 수 있습니다.

*   **페이지 이동**: 다음/이전 페이지로 이동하거나 특정 페이지 번호로 바로 이동할 수 있습니다.
*   **페이지당 행 수**: 한 페이지에 표시될 사용자 수를 10, 20, 50, 100개 중에서 선택할 수 있습니다.

### 2.5. 사용자 작업 (상세 보기 및 액션 메뉴)

각 사용자 행의 작업 열에 있는 아이콘을 통해 다음과 같은 작업을 수행할 수 있습니다.

*   **상세 보기**: 사용자 정보 상세 다이얼로그를 열어 해당 사용자의 상세 정보를 확인합니다.
*   **더 많은 작업 (점 세 개 아이콘)**: 클릭 시 드롭다운 메뉴가 나타나며, 다음 작업을 수행할 수 있습니다.
    *   **비활성화/활성화**: 사용자의 계정 상태를 활성 또는 비활성으로 전환합니다. 비활성화된 사용자는 시스템에 로그인할 수 없습니다.

### 2.6. 로딩 및 오류 처리

*   **로딩 표시**: 사용자 목록을 불러오는 동안 로딩 스피너가 표시됩니다.
*   **오류 메시지**: 데이터 로딩 중 오류가 발생하면 오류 메시지가 표시되며, 다시 시도할 수 있는 옵션이 제공될 수 있습니다.
*   **검색 결과 없음**: 검색 또는 필터 조건에 맞는 사용자가 없을 경우 해당 메시지가 표시됩니다.

## 3. 사용자 관리 페이지 사용 방법

1.  **페이지 접근**: 네비게이션 바에서 "사용자 관리" 버튼을 클릭하여 사용자 관리 페이지로 이동합니다.
2.  **사용자 검색 및 필터링**: 툴바의 검색 필드와 역할/상태 필터를 사용하여 원하는 사용자를 찾습니다.
3.  **사용자 목록 탐색**: 페이지네이션 컨트롤을 사용하여 목록을 탐색하거나, 열 헤더를 클릭하여 정렬합니다.
4.  **사용자 정보 확인**: 테이블에서 원하는 사용자의 정보를 확인합니다.
5.  **사용자 작업 수행**: 각 사용자 행의 작업 열에 있는 아이콘을 클릭하여 상세 정보를 보거나, 액션 메뉴를 통해 활성/비활성 상태를 변경합니다.

## 4. 참고 사항

*   사용자 역할 변경과 같은 추가적인 관리 기능은 사용자 상세 다이얼로그 내에서 제공될 수 있습니다.
*   데이터 내보내기 기능은 대량의 사용자 정보를 분석하거나 백업할 때 유용합니다.
