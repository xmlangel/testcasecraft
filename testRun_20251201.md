# 1. 전체 패키지 경로로 실행
./gradlew test --tests "com.testcase.testcasemanagement.api.AllApiComprehensiveTest"

# 2. 특정 테스트 메서드만 실행
./gradlew test --tests "com.testcase.testcasemanagement.api.AllApiComprehensiveTest.testUpdatePreviousTestResult"
./gradlew test --tests "com.testcase.testcasemanagement.api.AllApiComprehensiveTest.testDeletePreviousTestResult"
./gradlew test --tests "com.testcase.testcasemanagement.api.AllApiComprehensiveTest.testUpdatePreviousTestResultUnauthorized"

# 3. 패턴으로 실행
./gradlew test --tests "*AllApiComprehensiveTest"
또는 TestNG 그룹으로 실행하려면:

bash
# build.gradle에 TestNG 그룹 설정이 있다면
./gradlew test -Dgroups=api-comprehensive-test
추천 방법은 전체 패키지 경로를 사용하는 것입니다:

bash
./gradlew test --tests "com.testcase.testcasemanagement.api.AllApiComprehensiveTest"
이렇게 실행하시면 추가한 3개의 테스트를 포함한 모든 AllApiComprehensiveTest가 실행됩니다! 🚀


./gradlew apiComprehensiveTest
