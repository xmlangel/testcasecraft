### 4. passwordService.js 변환

**파일**: `src/main/frontend/src/services/passwordService.js`

**변경 사항**:
- ✅ `import axios from 'axios'` → `import axiosInstance from '../utils/axiosInstance.js'`
- ✅ `getAuthHeaders()` 메서드 제거
- ✅ 본인 비밀번호 변경 API 변환
- ✅ 관리자용 비밀번호 변경 API 변환

**API 호출**:
1. `PUT /api/auth/change-password` - 본인 비밀번호 변경
2. `PUT /api/admin/users/{userId}/password` - 관리자용 비밀번호 변경

**Before**:
```javascript
const response = await axios.put(url, data, {
  headers: {
    'Content-Type': 'application/json',
    ...this.getAuthHeaders()
  }
});
```

**After**:
```javascript
const response = await axiosInstance.put(url, data);
// Authorization 헤더는 interceptor가 자동으로 추가
// Content-Type은 axiosInstance가 자동으로 application/json 설정
```
