# 대시보드 사용 매뉴얼 (프로젝트 대시보드)

## 1. 개요

이 문서는 테스트케이스 관리 시스템의 프로젝트 대시보드 사용 방법을 안내합니다. 프로젝트 대시보드는 선택된 프로젝트의 테스트 결과 및 통계를 한눈에 파악할 수 있도록 다양한 차트와 요약 정보를 제공합니다.

## 2. 주요 기능 및 구성 요소

### 2.1. 프로젝트 정보 요약

대시보드 상단에는 현재 활성화된 프로젝트의 핵심 정보가 표시됩니다.

- **프로젝트 이름**: 현재 보고 있는 프로젝트의 이름입니다.
- **총 테스트케이스**: 프로젝트에 등록된 전체 테스트케이스의 수입니다.
- **프로젝트 멤버**: 프로젝트에 참여하고 있는 멤버의 수입니다.
- **최근 업데이트**: 대시보드 데이터가 마지막으로 업데이트된 시간을 표시합니다.
- **새로고침 버튼**: 대시보드 데이터를 수동으로 새로고침하여 최신 정보를 반영합니다.

### 2.2. 로딩 및 오류 처리

- **로딩 표시**: 데이터를 불러오는 동안 로딩 메시지가 표시됩니다.
- **오류 메시지**: API 호출 실패, 네트워크 문제, 인증 오류 등 발생 시 상세한 오류 메시지와 함께 해결 방법(예: 다시 시도, 로그인 페이지로 이동)을 안내합니다.
- **데이터 없음**: 프로젝트에 대시보드에 표시할 데이터가 없거나, 테스트 결과가 없는 경우 해당 메시지가 표시됩니다.

### 2.3. 핵심 지표 및 차트

대시보드는 프로젝트의 테스트 현황을 시각적으로 보여주는 다양한 차트와 요약 패널로 구성됩니다.

- **최근 테스트케이스 결과 (파이 차트)**:

  - **기능**: 프로젝트의 가장 최근 테스트 실행 결과(성공, 실패, 차단됨, 건너뜀, 미실행)의 분포를 파이 차트로 시각화합니다.
  - **주요 지표**: 전체 완료율(성공률)을 큰 숫자로 표시하며, 각 결과 유형별 건수를 함께 보여줍니다.

- **테스트케이스 결과 추이 (라인 차트)**:

  - **기능**: 지난 15일 동안의 테스트 결과(성공, 실패, 차단됨, 미실행) 변화 추이를 라인 차트로 보여줍니다.
  - **활용**: 시간 경과에 따른 테스트 품질 변화를 파악하고, 특정 시점의 이슈 발생 여부를 추적할 수 있습니다.

- **오픈 테스트런별 테스트케이스 결과 (가로 막대 차트)**:

  - **기능**: 현재 진행 중인(오픈된) 테스트런별로 테스트케이스 결과(성공, 미실행, 실패, 차단됨)를 스택형 막대 차트로 보여줍니다.
  - **활용**: 각 테스트런의 진행 상황과 결과 분포를 한눈에 파악할 수 있습니다.

- **오픈 테스트런 담당자별 테스트케이스 결과 (가로 막대 차트)**:

  - **기능**: 현재 진행 중인 테스트런에서 각 담당자(Assignee)별 테스트케이스 결과(성공, 미실행, 실패, 차단됨)를 스택형 막대 차트로 보여줍니다.
  - **활용**: 담당자별 작업 부하 및 완료율을 비교하고, 특정 담당자의 병목 현상을 식별하는 데 도움을 줍니다.

- **테스트 플랜별 최근 테스트 결과 (목록)**:

  - **기능**: 드롭다운 메뉴를 통해 특정 테스트 플랜을 선택하면, 해당 플랜의 최근 테스트 실행 결과 목록을 보여줍니다.
  - **활용**: 특정 테스트 플랜의 최신 실행 결과를 상세하게 검토할 수 있습니다.

- **오픈 테스트런 미실행 테스트케이스 추이 (라인 차트)**:
  - **기능**: 지난 15일 동안 오픈된 테스트런 내에서 미실행(NOTRUN) 상태의 테스트케이스 수 변화 추이를 라인 차트로 보여줍니다.
  - **활용**: 미실행 테스트케이스의 누적 현황을 파악하고, 테스트 진행의 지연 여부를 판단하는 데 사용됩니다.

## 3. 대시보드 사용 방법

1.  **페이지 접근**: 프로젝트를 선택한 후, 메인 화면의 대시보드 탭(기본 탭)을 통해 접근하거나, 네비게이션 바에서 "대시보드" 버튼을 클릭하여 접근합니다.
2.  **정보 확인**: 각 차트와 요약 패널을 통해 프로젝트의 전반적인 테스트 현황을 파악합니다.
3.  **데이터 새로고침**: 최신 데이터가 필요할 경우, 상단의 "새로고침" 칩을 클릭합니다.
4.  **테스트 플랜별 결과 조회**: "테스트 플랜별 최근 테스트 결과" 패널에서 드롭다운 메뉴를 사용하여 원하는 테스트 플랜을 선택하고 결과를 확인합니다.

## 4. 참고 사항

- 대시보드에 표시되는 데이터는 프로젝트의 실제 테스트 실행 결과에 기반합니다.
- 데이터 로딩에 시간이 걸릴 수 있으며, 네트워크 상태나 서버 응답에 따라 지연될 수 있습니다.
- 오류 발생 시 표시되는 메시지를 참고하여 문제를 해결하거나 관리자에게 문의하십시오.
