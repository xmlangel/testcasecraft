# 릴리스 노트 - v1.0.27

## 🚀 주요 변경 사항

### ⚡ 성능 최적화 (성능 대폭 개선)
- **API 호출 중복 방지**: 프로젝트 정보, JUnit 통계, 사용자 정보 등 빈번한 API 호출에 캐싱 및 중복 실행 방지(Deduplication) 로직을 적용하여 대시보드 로딩 속도를 크게 향상시켰습니다.
- **RAG API 최적화**: RAG 관련 API 호출에 공유 캐시를 도입하여 불필요한 서버 부하를 줄였습니다.

### 📊 테스트 케이스 및 스프레드시트 UX
- **깜빡임 제거**: 스프레드시트 업데이트 및 테스트 케이스 트리 동기화 시 발생하는 화면 깜빡임을 대폭 개선하여 부드러운 사용자 경험을 제공합니다.
- **상태 관리 최적화**: `updateTestCaseLocal` 도입을 통해 불필요한 서버 재조회 없이 클라이언트 상태를 즉시 반영하도록 리팩토링했습니다.
- **전체 화면 모드 지원**: 데이터 편집 시 몰입감을 높이기 위해 스프레드시트 영역의 전체 화면 전환 기능을 추가했습니다.

### 🛠 기능 개선
- **[fix] 지라 상태 업데이트 오류 수정**: 지라 연동 시 상태 정보가 즉시 반영되지 않던 문제를 해결했습니다.
- **[feat] 다국어 확대 적용**: 테스트 결과 및 UI 주요 영역에 누락되었던 다국어 번역을 추가 적용했습니다.
