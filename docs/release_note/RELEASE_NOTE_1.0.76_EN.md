# Release Note - v1.0.76

## [1.0.76] - 2026-05-14

Hello! In this 1.0.76 update, we have introduced a new design system, expanded AI agent capabilities, and enhanced multilingual support.

### Major Changes

#### 🎨 Introduction of New Design System (CreateSpace)
- **Added New Stylesheet**: Applied the new design system via `createspace.css`.
- **Improved UI Components**: Updated `UserProfileDialog` and `ThemeContext` to improve the user interface and theme switching functionality.

#### 🤖 Expansion of AI Agents and Skills
- **Added Specialized Agents**: Added specialized Claude agents for Design System QA, i18n key classification/writing, and profile UI extension.
- **Advanced Skill System**: Added various skills (SKILL.md) for agents to improve automation capabilities.

#### 🌐 Enhanced Multilingual (i18n) Support
- **Updated Translation Data**: Added and modified multilingual keys and Korean/English translation data for organization and user management.
