# 릴리스 노트 - v1.0.29

## 🚀 주요 변경 사항

### 🔐 인증 및 보안
- **세션 만료 알림 개선**: 세션 만료 시 단순히 로그아웃되는 대신, 만료 원인을 보여주는 전용 다이얼로그를 추가하여 사용자 경험을 개선했습니다.

### 🛠 시스템 및 성능
- **테스크 실행 폼 디버그 로깅**: 문제 발생 시 원인 파악을 용이하게 하기 위해 테스트 실행 폼에 상세 디버그 로그를 추가했습니다.
- **버전 업데이트**: 프론트엔드, 백엔드 및 Docker 설정을 포함한 전체 애플리케이션 버전을 1.0.29로 갱신했습니다.

### 🐛 버그 수정 ([fix])
- **[fix] 세션 만료 대화상자 오류수정**: 특정 상황에서 세션 만료 대화상자가 정상적으로 표시되지 않거나 오류가 발생하던 문제를 해결했습니다.
