# 릴리스 노트 - v1.0.32

## 🚀 주요 변경 사항

### 📊 UI/UX 및 디자인
- **스프레드시트 타이틀 고정**: 테스트 케이스 스프레드시트 화면에서 스크롤 시에도 헤더(타이틀)가 상단에 고정되도록 개선하여 대량 데이터 편집 시 가독성을 높였습니다.
- **다크 모드 스타일 강화**: 버전 비교 상세 화면 및 주요 UI 컴포넌트에 대한 다크 모드 스타일링을 정교하게 다듬어 시각적 통일성을 확보했습니다.

### 🏷 가시성 개선
- **디스플레이 ID 노출**: 테스트 케이스 상세 보기, 결과 헤더, 테스크 실행 헤더 등 주요 화면에 시스템 고유 ID 대신 사용자가 인지하기 쉬운 `displayId`를 노출하도록 변경했습니다.
