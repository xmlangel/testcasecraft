# 릴리즈 노트 - v1.0.42

## 🚀 주요 변경사항

### 📝 자동화테스트 노트 및 실행 편의성 개선
- **노트 컬럼 추가**: 자동화테스트 결과 상세 화면에 전용 노트 컬럼을 추가해 검토 속도를 높였습니다.
- **이전 노트 재사용**: 이전 테스트 케이스 실행 노트를 조회해 현재 실행 노트로 복사할 수 있는 기능을 추가했습니다.
- **노트 통계 제공**: 자동화테스트 결과 데이터에 `notesCount` 필드를 추가하고 대시보드에 노출했습니다.
- **상세 탭 새로고침 지원**: 편집 이후 자동화테스트 상세 탭이 최신 데이터로 갱신되도록 새로고침 동작을 구현했습니다.

### 🤖 RAG 기능 제어 및 운영 안정성 강화
- **RAG UI 토글 연동**: RAG 활성화/비활성화 상태에 따라 UI가 동적으로 동작하도록 개선했습니다.
- **스케줄러 자동 관리**: RAG 상태 변경 시 관련 스케줄러를 자동으로 관리하는 로직을 추가했습니다.
- **비활성 상태 보호**: RAG가 비활성화된 경우 관련 스케줄러가 활성화되지 않도록 보호 로직을 적용했습니다.
- **사용자 안내 개선**: RAG 활성/비활성 상태에 대한 사용자 안내 메시지를 보강했습니다.

### 🎨 자동화테스트 상세 UI/UX 개선
- **반응형 페이지 크기**: 화면 높이에 맞춰 JUnit 상세 페이지 크기가 동적으로 계산되도록 구현했습니다.
- **긴 텍스트 가독성 개선**: 결과 상세 테이블의 긴 텍스트에 말줄임 및 툴팁을 적용했습니다.
- **표시 포맷 개선**: `word-break` 동작과 트레이스 로그 메시지 포맷을 개선해 가독성을 높였습니다.

## 🛠 빌드, 테스트, 개발 경험 개선
- **로컬 AMD64 빌드 스크립트 추가**: `build-local-amd64.sh`와 관련 빌드 가이드를 추가했습니다.
- **E2E 테스트 인프라 정비**: `AutomationPage` 픽스처를 추가하고 회귀 테스트 스펙 구성을 정리했습니다.
- **자격 증명 설정 리팩토링**: E2E 테스트 유지보수를 위해 관리자 로그인 자격 증명을 외부 설정으로 분리했습니다.
- **스킬/가이드 업데이트**: Playwright E2E 스킬 정의를 추가하고 E2E 문서를 최신화했습니다.

## 📌 기타 개선사항
- **프론트엔드 의존성 업데이트**: 프론트엔드 의존성 패키지를 업데이트했습니다.
- **i18n 키 업데이트**: 다국어 키를 추가 및 정비했습니다.
- **문서 및 파일 정리**: README 브랜딩 섹션을 업데이트하고 미사용 파일을 정리했습니다.
