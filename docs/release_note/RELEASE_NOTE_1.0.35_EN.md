# Release Notes - v1.0.35

## 🚀 Key Changes

### 📈 UI Innovation & Visibility Improvement (JUnit Results)

- **Accordion-based Navigation**: Replaced the existing tab-based sections in the JUnit result screen with expandable Accordion presets, allowing users to view more information efficiently within limited space.

- **Automated State Management**: Added functionality to persist Accordion expanded/collapsed states and completed all necessary i18n translations.

- **Docker Build Path Synchronization**: Unified the `docker-compose-dev-spring` directory name to `docker-compose-build` and updated all related paths in build scripts.
- **Unified Version Bump**: Incremented the application version to 1.0.35 across frontend, backend, and Docker configurations.

### 📚 Documentation & Guidelines
