# 릴리즈 노트 - v1.0.43

## 🚀 주요 변경사항

### 🏗️ CI/CD 및 Docker 빌드 프로세스 개선
- **GitHub Action 워크플로우 강화**: Docker 빌드 및 푸시 과정에서의 안정성을 높이고, `contents: write` 권한 부여를 통해 자동화된 릴리즈 처리가 가능하도록 개선했습니다.
- **빌드 스크립트 옵션 확장**: `build-and-push-multiplatform.sh` 등 주요 빌드 스크립트에 `--skip-tag-verify` 옵션을 추가하여 유연한 태그 관리가 가능해졌습니다.
- **프리릴리즈 지원 개선**: 프리릴리즈 버전에 대한 파싱 및 유효성 검사 로직을 강화하고, `latest` 태그 지정 방식을 최적화하여 버전 관리의 정확도를 높였습니다.
- **경로 관리 최적화**: 빌드 경로 및 JAR 파일 관리 시 스크립트 디렉토리 경로를 명시적으로 사용하도록 리팩토링하여 빌드 환경의 독립성을 강화했습니다.
- UI 테스트 및 API 테스트 추가
