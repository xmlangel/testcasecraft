# Release Note - v1.0.70

## [1.0.70] - 2026-04-17

안녕하세요! 이번 1.0.70 업데이트에서는 **시스템을 더 튼튼하게** 만들고, **한국어와 영어가 자연스럽게 나오도록** 꼼꼼하게 챙겼습니다. 또한 **화면을 더 예쁘고 쓰기 편하게** 바꾸는 등 사용자 여러분의 편의를 위해 많은 노력을 기울였습니다.

### 주요 변경 사항

#### ✨ 한국어와 영어가 더 자연스러워졌어요
- **날짜 표시가 자연스러워요**: 대시보드나 리스트에서 날짜가 어색하지 않게 사용자 설정에 맞춰 표시되도록 개선했습니다.
- **곳곳에 빠진 번역을 채웠어요**: 테스트 케이스 이름을 바꾸거나 프로필을 볼 때 한국어와 영어가 정확하게 표시됩니다.
- **도움말도 영어로 볼 수 있어요**: 구글 시트 연동 가이드를 영문으로도 추가해서, 영어가 편하신 분들도 쉽게 도움을 받으실 수 있습니다.

#### 🎨 화면이 더 예뻐지고 쓰기 편해졌어요
- **구글 설정 화면이 깔끔해졌어요**: 구글 시트 연동 설정 화면을 더 보기 좋고 정렬된 디자인으로 새롭게 단장했습니다.
- **원하는 곳으로 바로 가요**: 내 정보 화면에서 구글 설정 탭 등으로 한 번에 이동할 수 있는 버튼을 넣었습니다.
- **번호(ID)를 바로 확인해요**: 테스트 케이스의 고유 번호를 화면에서 바로 볼 수 있게 해서 작업 효율을 높였습니다.

#### 🚀 시스템이 더 똑똑하고 안정적으로 변했어요
- **불필요한 에러 메시지를 줄였어요**: 설정이 안 되어 있을 때 나오던 딱딱한 에러 메시지들을 정리해서 화면이 더 깨끗해졌습니다.
- **구글 시트 연동이 더 똑똑해요**: 주소가 맞는지 미리 확인해주고, 마지막에 썼던 설정을 기억해서 다음에도 바로 쓸 수 있게 했습니다.
- **작동이 더 매끄러워요**: 내부 구조를 튼튼하게 고쳐서 버튼을 누르거나 화면을 이동할 때 멈춤 없이 부드럽게 작동합니다.

#### 🛠️ 그 외에 고친 것들
- 화면에서 글자가 겹치거나 잘 안 보이던 부분들을 수정했습니다.
- 프로그램이 갑자기 멈출 수 있는 작은 버그들을 여럿 찾아내어 해결했습니다.
- 소중한 데이터를 더 안전하게 보관할 수 있도록 보이지 않는 곳의 코드들을 다듬었습니다.
