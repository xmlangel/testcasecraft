# Release Note - v1.0.66

## [1.0.66] - 2026-04-14

v1.0.65 이후의 주요 기능 개선 및 시스템 최적화 사항을 포함한 릴리즈 노트입니다.

### 주요 변경 사항

#### 🚀 자동화 테스트 결과 인피니티 스크롤 도입 (Infinity Scroll for Automation Results)
- **성능 및 UX 최적화**: 자동화 테스트 결과 목록에 인피니티 스크롤을 적용하여 대용량 데이터를 부드럽게 탐색할 수 있도록 개선하였습니다.
- **메모리 효율성**: 한꺼번에 많은 데이터를 로드하지 않고 스크롤 시점에 필요한 데이터를 호출함으로써 클라이언트 메모리 부하를 줄였습니다.

#### 🛠️ 빌드 시스템 및 런타임 최적화 (Build & Runtime Optimization)
- **Gradle 태스크 고도화**: `bootRun` 태스크가 `classes` 태스크에 명시적으로 의존하도록 하고, 런타임 클래스패스를 정교하게 설정하여 실행 안정성을 높였습니다.
- **개발 환경 개선**: 원격 실행 관련 태스크 구성을 최적화하여 개발 생산성을 향상시켰습니다.

#### ✨ UI 상세 페이지 개선 (UI Enhancement)
- **MUI 컴포넌트 추가 및 최적화**: `JunitResultDetail` 페이지에서 Material-UI 컴포넌트(`FormControl` 등) 사용을 정교화하고 레이아웃을 개선하였습니다.

#### 🔒 보안 강화 및 종속성 업데이트 (Security & Dependency Updates)
- **보안 패치**: 베이스 이미지의 패키지를 최신 상태로 업그레이드하여 보안 취약점을 해결하고 시스템 보안을 강화하였습니다.
- **라이브러리 업데이트**: 프로젝트 종속성 버전을 최신화하여 시스템 안정성 및 호환성을 확보하였습니다.

#### 🐜 주요 버그 수정 (Bug Fixes)
- **테스트 결과 검증 로직 수정**: `TestResultReportServiceMockTest`를 최적화된 서비스 로직에 맞게 업데이트하여 테스트 신뢰도를 높였습니다.
