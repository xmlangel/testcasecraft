# 릴리스 노트 - v1.0.26

## 🚀 주요 변경 사항

### 📈 UI 및 사용자 편의성

- **페이지네이션 개선**: 테스트 실행 목록 및 자동화 결과 테이블의 페이지네이션 로직을 보완하여 데이터가 없는 중간 페이지 처리를 안정화했습니다.
- **직관적인 상태 표시**: 테스트 결과 상태별 컬러 코딩을 다듬어 가시성을 높였습니다.

### 🛠 시스템 기반 구축

- **OpenAPI 동적 버전 관리**: BuildProperties를 활용하여 Swagger UI 등에 노출되는 OpenAPI 버전을 자동으로 설정하도록 개선했습니다.
- **라이선스 및 문서 정비**: 프로젝트 라이선스를 Apache 2.0으로 공식 지정하고, Swagger API 문서의 연락처와 상세 설명을 현행화했습니다.

### 🐛 주요 버그 수정 ([fix])

- **[fix] 독립 프로젝트 표시 문제**: 특정 상황에서 독립 프로젝트가 대시보드에 나타나지 않던 오류를 해결했습니다.
- **[feat] Spring Boot DevTools 도입**: 개발 효율성을 위해 DevTools 설정을 추가하고 전역 에러 핸들링 시 스택 트레이스 노출 방식을 개선했습니다.
