# Release Note - v1.0.81

## [1.0.81] - 2026-06-06

안녕하세요! 이번 1.0.81 업데이트는 **다국어(한/영) 경험의 완성**에 집중하였습니다. 영어 모드에서 한국어 문구가 그대로 노출되던 문제를 전수 조사하여 **누락 번역 1,196건을 일괄 보강**하였고, 그 위에서 **영문 사용자 매뉴얼**을 신설하였습니다. 한국어 매뉴얼도 v1.0.80 기준으로 전수 검증·보강되었습니다.

### 주요 변경 사항

#### 🌐 한/영 전환 전수 보강 (핵심)

* 프런트엔드 전체를 전수 감사하여 **영어 모드에서 한국어가 노출되던 문구를 일괄 해소**하였습니다.
  - 번역 키는 있으나 영어 번역이 없던 482건 → 영어 번역 시드
  - 번역 시스템(t()) 밖에 하드코딩되어 있던 한국어 약 800건(99개 화면 파일) → 번역 키로 전환 + 한/영 번역 713건 신설
* 필터·테스트 실행 목록·"총 N건" 카운트·태그 입력 안내·JIRA 연동 라벨·툴팁·차트 포맷 등 사용자가 보고하신 문구가 모두 영어로 전환됩니다.
* 신규 설치에도 적용되도록 번역 시드 클래스(`I18nGapKeysInitializer`, `I18nHardcodedKeysInitializer` 외 4종)를 추가하였습니다 — 앱 시작 시 멱등 시드.
* 번역 누락을 상시 검출할 수 있는 스캐너 `scripts/i18n_scan.py` 를 추가하였습니다 (t() 밖 한국어 문자열 검출, CI 연동 가능).

#### 📘 영문 사용자 매뉴얼 신설

* `docs/manual/new/USER_MANUAL_EN.md` — 한국어 매뉴얼과 18개 섹션·소제목 49개가 1:1 대응하는 완역본입니다 (ISO/IEC/IEEE 26514 테크니컬 라이팅 톤).
* 스크린샷 68장을 **영어 UI + 영문 샘플 데이터(ShopFlow EN)** 로 전부 신규 캡처하여 `docs/manual/new/images_en/` 에 수록하였습니다. 회원가입·프로젝트 생성·프로필 7탭 같은 다이얼로그 화면도 인터랙션 자동화로 캡처.
* 한↔영 매뉴얼 머리말에 상호 링크를 두어 언어 전환이 쉽습니다.

#### 📖 한국어 사용자 매뉴얼 전수 보강 (v1.0.80 기준)

* 코드 ↔ 매뉴얼 전수 대조(라우트 20종·수치 11건·릴리즈 노트 반영도·이미지 98장)를 수행하고 불일치를 교정하였습니다.
  - §8-1 결과 입력 화면(P/F/B/N 4종) · §8-2 자동 저장 동작(v1.0.80 재설계) 신설
  - §9 결과 보고서 내보내기 형식 교정 (실제 제공: Excel / PDF / CSV)
  - §17-9·§18-4 프로젝트 역할 6종 교정 (PROJECT_MANAGER / LEAD_DEVELOPER / DEVELOPER / TESTER / CONTRIBUTOR / VIEWER)
  - 폼 1단 풀폭 레이아웃(v1.0.80)·에디터 자동 높이(v1.0.79) 변경 메모 반영
* 머리말에 **"이 매뉴얼을 읽는 방법"**(독자별 시작 섹션 + 표기 규칙)을 추가하였습니다 — IEC/IEEE 82079-1 사용 정보 표준 기준.
* 트리 우클릭 메뉴·폴더 구조·필드 표시·폼 메타데이터 화면 캡처 4장을 신규 수록하였습니다.

#### 🎨 투명 로고 적용

* 헤더 로고를 다크/라이트 분기 JPG 2종에서 **단일 투명 PNG** 로 교체하여 어떤 테마에서도 자연스럽게 표시됩니다.

### 💡 사용자 조치 가이드

#### 영어 화면으로 사용하기
1. 헤더 우상단 **아바타 → 프로필 → 언어 설정** 탭에서 English 를 선택합니다.
2. 즉시 화면 전체가 영어로 전환됩니다. 이전 버전에서 한국어로 남아 있던 문구들이 이번 업데이트로 모두 번역됩니다.
3. 영어 매뉴얼은 `docs/manual/new/USER_MANUAL_EN.md` 에서 확인하세요.

#### 아직 한국어로 보이는 항목
* 사용자가 입력한 데이터(프로젝트·케이스 이름 등)는 번역 대상이 아닙니다.
* 일부 시스템 데이터(스케줄러 작업 이름)와 날짜의 오전/오후 표기는 다음 업데이트에서 개선 예정입니다.
