# 릴리스 노트 - v1.0.34

## 🚀 주요 변경 사항

### 🧪 테스트 자동화 및 인프라
- **인증 API 테스트 강화**: JSON 스키마 검증을 포함한 포괄적인 인증 API 테스트 세트를 추가하고, 테스트 설정을 기존 `.properties`에서 `.yml` 형식으로 마이그레이션하여 관리 효율성을 높였습니다.
- **데이터 초기화 오류 수정**: `DataInitializer`에서 발생하던 외래 키 제약 조건 위반 문제를 해결하고 초기 데이터 생성 로직의 안정성을 강화했습니다.

### 🎨 디자인 최적화
- **JUnit 결과 패널 색상 조정**: 다크 모드 환경에서의 시인성을 위해 JUnit 테스트 케이스 상세 패널의 배경색과 텍스트 색상을 최적화했습니다.
- **인프라 디렉토리 통합**: 개발 및 빌드 환경의 일관성을 위해 Docker 관련 디렉토리 구조를 개선했습니다.
