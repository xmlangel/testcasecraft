# Release Note - v1.0.55

## [1.0.55] - 2026-03-30

### 주요 변경 사항

#### 🚀 주요 기능 및 개선 사항 (Features & Enhancements)

- **Ollama 기반 로컬 코드 리뷰 시스템 도입**:

  - **다중 모델 리뷰 지원**: `pre-commit` 단계에서 DeepSeek, Qwen 등 다양한 로컬 LLM을 활용한 자동 코드 리뷰 기능을 추가했습니다.
  - **실시간 스트리밍 및 번역**: 리뷰 결과를 실시간으로 확인하고, 영어 출력을 한국어로 자동 번역하는 기능을 지원하여 한국어 사용자 편의성을 강화했습니다.
  - **유연한 리뷰 대상 지정**: `--file` 플래그를 통해 특정 파일만 선택적으로 리뷰할 수 있는 기능을 추가했습니다.

- **프론트엔드 UI/UX 개선**:
  - **탐색 세션 탭 동적 제어**: 환경 변수 설정을 통해 '탐색 세션' 탭의 노출 여부를 동적으로 관리할 수 있도록 개선했습니다.
  - **차터(Charter) 생성 로직 고도화**: 탐색적 테스트를 위한 차터 생성 기능을 최적화하여 보다 정교한 품질 관리가 가능해졌습니다.

#### 🛠️ 인프라 및 개발 환경 (Infrastructure & DevEnv)

- **코드 품질 관리 자동화 (Pre-commit)**:
  - **통합 린트/포맷터 설정**: Python(Black, Flake8), JavaScript(ESLint, Prettier), Java(Google Java Format)용 `pre-commit` 훅을 전면 도입하여 코드 일관성을 자동화했습니다.
  - **보안 및 예외 처리 강화**: 리뷰 스크립트의 에러 핸들링과 보안 검증 로직을 개선하여 안정성을 높였습니다.
