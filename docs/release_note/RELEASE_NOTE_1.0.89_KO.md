# Release Note - v1.0.89

## [1.0.89] - 2026-06-15

안녕하세요! 1.0.89는 **프로젝트 선택 화면의 레이아웃을 다듬은** 소규모 UI 업데이트입니다. 상단 헤더 영역이 차지하던 세로 공간을 줄여, 프로젝트 카드가 화면 위쪽에서 더 빨리 눈에 들어오도록 했습니다. 기능 변화는 없습니다.

### 주요 변경 사항

#### 🎨 프로젝트 선택 헤더 영역 압축 (UI)

프로젝트 선택 화면 상단의 "프로젝트 선택 / 프로젝트 관리 / 새 프로젝트 생성" 영역이 불필요하게 넓던 문제를 정리했습니다.

* **"프로젝트 선택" 라벨**: 큰 제목(`h5`)에서 작은 캡션 스타일로 축소하고 상하 여백을 최소화했습니다.
* **"프로젝트 관리" 제목**: `h4` → `subtitle1`(굵게)로 축소하고 하단 여백을 줄였습니다.
* **"새 프로젝트 생성" 버튼**: `small` 크기로 조정해 헤더 높이에 맞췄습니다.

결과적으로 헤더가 차지하던 세로 공간이 크게 줄어 프로젝트 카드 목록이 위로 올라옵니다.

### 업그레이드 안내

* DB 마이그레이션 없음 — 이미지 교체만으로 적용됩니다.
* 데이터·기능 동작에는 변화가 없으며, 프로젝트 선택 화면의 상단 레이아웃만 더 조밀해집니다.
* 1.0.88의 초기 화면 진입 성능 개선은 [RELEASE_NOTE_1.0.88_KO.md](RELEASE_NOTE_1.0.88_KO.md)를 참고하세요.
