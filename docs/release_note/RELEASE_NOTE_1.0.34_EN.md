# Release Notes - v1.0.34

## 🚀 Key Changes

### 🧪 Test Automation & Infrastructure

- **Enhanced Auth API Tests**: Added a comprehensive set of authentication API tests including JSON schema validation and migrated test configuration from `.properties` to `.yml` for better manageability.
- **Data Initialization Fix**: Resolved foreign key constraint violations in the `DataInitializer` and strengthened the reliability of the initial data generation logic.

### 🎨 Design Optimization

- **JUnit Result Panel Color Adjustment**: Optimized background and text colors for the JUnit test case detail panel for better visibility in dark mode.
- **Infrastructure Directory Integration**: Improved the Docker-related directory structure for better consistency between development and build environments.
