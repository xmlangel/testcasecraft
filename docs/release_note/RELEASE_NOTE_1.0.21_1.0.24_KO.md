# 릴리스 노트 - v1.0.21 ~ v1.0.24 (플랫폼 확장기)

## 🚀 주요 변경 사항

### 🏢 엔터티 및 데이터 구조화 (v1.0.24)
- **조직 데이터 초기화 강화**: 프로젝트 초기 기동 시 조직 및 부서 데이터를 더 체계적으로 생성하도록 초기화 로직을 업데이트했습니다.

### 🧪 테스트 관리 고도화 (v1.0.21 ~ v1.0.23)
- **계층형 트리 가시성 개선**: 테스트 케이스 트리의 노드 순서와 폴더별 테스트 케이스 개수 표시 가시성을 높였습니다.
- **스프레드시트 입력 유연성**: 폴더나 테스트 케이스가 하나도 없는 상태에서도 스프레드시트를 통해 대량 입력이 가능하도록 최적화했습니다.

### 🛠 UI/UX 편의 기능
- **리스트 정렬 기준 변경 (v1.0.22)**: 프로젝트 및 테스트 플랜 목록의 정렬 기준을 기존 '수정순'에서 '생성순'으로 변경하여 데이터 일관성을 높였습니다.
- **일괄 선택 및 작업 기능**: 헤더에 '전체 선택' 체크박스를 통합하고 액션 버튼 레이아웃을 개선하여 대량 작업의 편의성을 높였습니다.
- **입력 폼 가이드**: 테스트 케이스 입력 시 폴더를 선택하지 않았을 경우 안내 문구를 추가하여 사용자 실수를 방지했습니다.

### 🐛 버그 수정 ([fix])
- **DB 초기화 오류 수정 (v1.0.21)**: 시스템 최초 기동 시 발생하던 데이터베이스 초기화 관련 오류를 해결했습니다.
- **트리 포커스 오류 수정 (v1.0.23)**: 테스트 케이스 트리에서 특정 노드 선택 시 포커스가 유실되던 문제를 수정했습니다.
