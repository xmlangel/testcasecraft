# Release Note - v1.0.94

## [1.0.94] - 2026-07-13

안녕하세요! 1.0.94는 최근 트리 리팩토링·인가 하드닝 이후 발생한 **테스트 케이스 트리 순서 변경 회귀 두 건을 바로잡는** 버그 수정 릴리즈입니다.

### 주요 변경 사항

#### 🔀 트리 순서 변경 회귀 수정 (수정)

* **드래그앤드롭 이동이 403으로 실패하던 문제 수정**: 트리 이동(DnD) 인가가 시스템 관리자를 제외하는 검사(`hasEditRole`)를 쓰고 있어, 테스트 케이스 생성·수정은 되는데 **드래그로 위치를 옮길 때만 403 권한 오류**가 났습니다. 이제 생성/수정/삭제와 동일한 표준 인가 검사(`canEditProject` = 시스템 ADMIN 또는 프로젝트 편집 역할)를 사용하도록 정렬해, **시스템 관리자도 정상적으로 이동**할 수 있습니다.
* **폴더 전용 트리에서 위/아래 순서 변경이 반영되지 않던 문제 수정**: 순서 변경 모드에서 위/아래 버튼이 화면에 보이지 않는 테스트 케이스와 자리를 바꿔, **폴더 순서가 시각적으로 바뀌지 않던** 문제를 수정했습니다. 폴더 전용 보기에서는 같은 종류(폴더)끼리만 순서를 교환하도록 했고, 전체 보기(폴더+케이스)의 기존 동작은 그대로 유지됩니다.

#### ✅ 회귀 방지 테스트 추가

* 트리 이동 인가 통합 테스트를 추가했습니다 — 시스템 관리자·프로젝트 편집 역할은 이동 허용(200), VIEWER/TESTER/비멤버는 거부(403).
* 순서 변경(위/아래) 프론트엔드 단위 테스트를 추가했습니다 — 폴더 전용 보기에서 케이스를 건너뛰고 폴더끼리 교환, 전체 보기의 기존 동작 보존, 조회 전용(VIEWER) 차단.

### 업그레이드 안내

* DB 마이그레이션 없음 — 이미지 교체만으로 적용됩니다.
* 1.0.93의 다른 프로젝트로 이동·복사, 인가 보안 하드닝은 [RELEASE_NOTE_1.0.93_KO.md](RELEASE_NOTE_1.0.93_KO.md)를 참고하세요.
