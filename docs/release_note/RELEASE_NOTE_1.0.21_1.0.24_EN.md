# Release Notes - v1.0.21 to v1.0.24 (Platform Expansion)

## 🚀 Key Changes

### 🏢 Entity & Data Structuring (v1.0.24)
- **Enhanced Organization Initialization**: Updated initialization logic to create more structured organization and department data on startup.

### 🧪 Advanced Test Management (v1.0.21 - v1.0.23)
- **Improved Tree Visibility**: Enhanced visibility for node order and test case counts within the hierarchical test case tree.
- **Spreadsheet Flexibility**: Optimized the spreadsheet to allow bulk input even when no folders or test cases exist initially.

### 🛠 UI/UX Enhancements
- **List Sorting Change (v1.0.22)**: Updated the sorting criteria for Project and Test Plan lists from 'Last Update' to 'Creation Date' for better data consistency.
- **Bulk Selection & Actions**: Integrated a 'Select All' checkbox into the header and improved action button layouts for better bulk operations.
- **Input Form Guidance**: Added guidance messages when no folder is selected during test case creation to prevent user errors.

### 🐛 Bug Fixes ([fix])
- **DB Initialization Fix (v1.0.21)**: Resolved errors related to database initialization during the first system startup.
- **Tree Focus Error Fix (v1.0.23)**: Fixed an issue where the focus would be lost within the test case tree when selecting certain nodes.
