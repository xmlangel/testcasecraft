# Release Note - v1.0.82

## [1.0.82] - 2026-06-06

안녕하세요! 이번 1.0.82 업데이트는 **테스트케이스 화면의 정보 밀도 개선**에 집중하였습니다. 트리는 폴더 구조 탐색에, 우측 목록은 케이스 열람에 집중하도록 역할을 나누어, 케이스가 많은 프로젝트에서도 전체 구조를 한눈에 파악할 수 있습니다.

### 주요 변경 사항

#### 🌳 폴더 전용 트리 (핵심)

* 테스트케이스 트리가 기본적으로 **폴더만 표시**합니다. 케이스는 우측의 케이스 목록에서 열람합니다.
  - 트리 헤더의 트리 모양 아이콘으로 기존 **폴더+케이스 혼합 모드**로 언제든 전환할 수 있으며, 선택은 브라우저에 저장되어 유지됩니다.
  - 테스트 플랜에서 케이스를 선택하는 화면에서는 항상 케이스가 포함된 전체 트리가 표시됩니다.
* 트리 상단에 **가상 노드** 2개가 고정 표시됩니다 — **모든 테스트케이스**(전체 개수), **폴더에 없는 테스트케이스**(폴더 미지정 개수). 클릭하면 해당 목록이 우측에 열립니다.
* 트리 헤더 아래 **폴더 필터** 검색창이 추가되었습니다. 폴더 이름 부분 일치로 트리를 추려 보여주고, 일치 항목의 상위 경로는 자동으로 펼쳐집니다.
* 트리 행 표시를 정리하였습니다 — 평소에는 **펼침 화살표 + 아이콘 + 이름 + 케이스 개수**만 보이고, 체크박스·드래그 핸들·버전 히스토리·더보기(⋮) 버튼은 행에 마우스를 올렸을 때 나타납니다. 케이스 ID는 이름에 마우스를 올리면 툴팁으로 확인합니다.

#### 📋 폴더 케이스 목록 (신규)

* 폴더를 선택하면 우측에 직속 하위 폴더와 **하위 폴더를 포함한 모든 케이스**가 **이름 → 설명 → 기대결과 → 우선순위** 컬럼의 표로 표시됩니다. 하위 폴더 소속 케이스에는 선택 폴더 기준 상대 경로의 **폴더** 컬럼이 붙습니다.
  - "모든 테스트케이스" / "폴더에 없는 테스트케이스" 목록에서는 맨 앞에 **폴더 경로** 컬럼이 추가되고, 경로를 클릭하면 해당 폴더로 이동합니다.
  - 텍스트가 100자를 넘으면 말줄임(…) 처리되고 마우스를 올리면 전체 내용이 툴팁으로 표시됩니다.
  - 기대결과는 케이스의 통합 기대결과를 우선 표시하고, 없으면 스텝별 기대결과를 번호를 매겨 보여줍니다.
* 목록 상단 폴더명 옆 **연필 아이콘**으로 폴더 정보(이름·설명·태그) 편집 폼에 진입하고, 저장 또는 "← 케이스 목록으로 돌아가기" 링크로 복귀합니다.
* 우측 본문 상단에 **경로 표시(브레드크럼)** 가 추가되었습니다 — `상위폴더 › 하위폴더 › 케이스명` 형태로 현재 위치를 보여주고, 상위 폴더를 클릭해 바로 이동합니다.

#### 🔢 탭 개수 배지

* **테스트케이스 / 테스트 플랜 / 테스트실행** 탭 라벨 옆에 각각의 개수가 배지로 표시됩니다.

#### 🐛 버그 수정

* 트리 우클릭(또는 ⋮) 메뉴에서 **[이름 변경]을 눌러도 다이얼로그가 열리지 않던 문제**를 수정하였습니다. 메뉴 항목 클릭 시 메뉴가 닫히지 않아 다이얼로그 표시가 차단되던 원인입니다.
* 도커 배포에서 **헤더 로고 이미지가 표시되지 않던 문제**를 수정하였습니다 (`.gitignore` 의 전역 `*.png` 규칙이 로고 파일을 제외시켜 빌드 산출물에서 누락).

#### ⚡ 성능·에셋

* 헤더 로고 PNG를 1.6MB → **47KB** 로 최적화하여 모든 페이지의 초기 로드를 가볍게 하였습니다.
* iOS 홈 화면 추가용 `logo192.png` 를 신설하고 PWA manifest 에 등록하였습니다.

#### 📖 매뉴얼·번역

* 사용자 매뉴얼(한/영) §3-1·§4·§5 를 본 개편에 맞춰 갱신하고, 신규 스크린샷 4장(폴더 전용 트리·폴더 케이스 목록·전체 케이스 목록·폴더 정보 편집)을 한/영 각각 수록하였습니다. §18 용어집에 신규 용어 4종(폴더 케이스 목록·가상 노드·폴더 필터·경로 표시)을 등재하였습니다.
* 신규 UI 문구 약 20종의 한/영 번역을 시드에 추가하였습니다 — 신규 설치에도 동일 적용.

### 업그레이드 안내

* 별도 마이그레이션 없이 이미지 교체만으로 적용됩니다. 신규 번역 키는 앱 시작 시 자동 시드됩니다.
* 트리 표시 모드 기본값이 "폴더만"으로 변경되었습니다. 기존 혼합 모드를 선호하시면 트리 헤더의 트리 모양 아이콘으로 전환하세요 (1회 설정 후 유지).
