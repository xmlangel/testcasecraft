# Release Note - v1.0.90

## [1.0.90] - 2026-06-16

안녕하세요! 1.0.90은 **테스트 결과·실행 화면의 마크다운 표시 품질**을 개선한 업데이트입니다. QA 총평·노트(비고)에서 마크다운이 어색하게 보이거나 과도한 빈 공간이 생기던 문제를 정리했고, 긴 노트도 스크롤 없이 한눈에 보이도록 했습니다. 프런트엔드 단위 테스트 인프라(vitest)도 신설했습니다.

### 주요 변경 사항

#### 🎨 마크다운 과도한 공백 제거 (수정)

QA 총평을 비롯해 마크다운으로 표시되는 여러 화면에서 문단·표 사이에 큰 빈 공간이 생기던 문제를 일괄 정리했습니다.

* **원인**: 마크다운 본문 전체에 `pre-wrap`(공백·줄바꿈 그대로 표시)이 걸려, 문단·표 사이의 보이지 않는 줄바꿈까지 빈 줄로 렌더링되었습니다.
* **수정**: 줄바꿈 보존을 문단·목록 내부로만 한정했습니다. 문단 안에서 직접 입력한 줄바꿈은 그대로 유지되고, 블록 사이의 불필요한 빈 줄은 사라집니다.
* **적용 범위**: QA 총평, 테스트 결과의 비고(노트), 이전 실행 결과·JIRA 이력의 노트, RAG 문서 청크·요약·분석 결과 화면 전반.

#### 📝 테스트 실행 노트 전체 표시 (개선)

* 테스트 실행 화면의 **노트(비고)** 가 고정 높이라 내용이 길면 미리보기에서 스크롤이 생기던 것을, 테스트 스텝처럼 **값이 있으면 스크롤 없이 전체가 보이도록** 개선했습니다. 편집·전체화면 모드에서는 기존 고정 높이를 유지합니다.

#### 🧪 프런트엔드 단위 테스트 도입 (내부 품질)

* vitest + Testing Library + jsdom 기반 단위 테스트 인프라를 신설하고, 위 마크다운/노트 변경에 대한 회귀 테스트(8개 파일·25개 케이스)를 추가했습니다. `npm test` 로 실행합니다.

### 업그레이드 안내

* DB 마이그레이션 없음 — 이미지 교체만으로 적용됩니다.
* 데이터·기능 동작에는 변화가 없으며, 마크다운 표시와 노트 영역의 화면 표현만 개선됩니다.
