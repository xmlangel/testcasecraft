# Release Note - v1.0.51

## [1.0.51] - 2026-03-26

### 주요 변경 사항

#### 🚀 주요 기능 및 개선 사항 (Features & Enhancements)
- **마크다운 노트 인라인 이미지 기능 강화 (ICT-386)**:
  - 마크다운 편집기 내에서 이미지 붙여넣기(Paste) 및 직접 업로드 기능을 추가하여 문서 작업 효율을 대폭 향상했습니다.
  - 이미지 업로드 완료 시 추가 다이얼로그 확인 없이 즉시 본문에 삽입되도록 프로세스를 최적화했습니다.
  - 테스트 케이스나 결과 삭제 시 연결된 인라인 이미지 파일도 함께 삭제되도록 파일 관리 로직을 개선했습니다.
- **시스템 안정성 및 아키텍처 개선**:
  - 프론트엔드 컨텍스트 참조 방식을 `useAuth`, `useI18n` 직접 호출 방식으로 리팩토링하여 컨텍스트 간 순환 참조(Circular Dependency) 문제를 근본적으로 해결하고 애플리케이션 초기화 속도를 개선했습니다.

#### 🐞 버그 수정 (Bug Fixes)
- **인라인 이미지 삽입 오류 수정**: 마크다운 노트에 이미지를 삽입할 때 발생하던 placeholder 교체 실패 버그를 수정했습니다.
- **코드 품질 강화**: 프론트엔드 컴포넌트와 백엔드 테스트 코드에서 불필요한 미사용 import를 전면 정리하였습니다.
