# Release Note - v1.0.48

## [1.0.48] - 2026-03-24

### 주요 변경 사항

#### 🚀 주요 기능 (Features)
- **계층적 폴더 통계 도입**: 대시보드에 폴더 구조를 반영한 상세 계층 통계 뷰를 추가하였습니다.
- **테스트 실행 횟수 계산**: 통계 및 리포트에서 각 테스트 케이스의 총 실행 횟수를 계산하여 표시하는 기능을 도입하였습니다.
- **대시보드 통계 상세 다이얼로그**: 대시보드 통계 카드에서 "미실행", "실패", "Jira 연동" 테스트 케이스 목록을 즉시 확인할 수 있는 모달 다이얼로그를 추가하였습니다.
- **리포트 범위 확장**: 폴더가 아닌 모든 테스트 케이스 타입이 테스트 결과 리포트에 누락 없이 포함되도록 개선하였습니다.

#### 🛠 개선 사항 (Improvements)
- **필터링 목록 페이지네이션**: 필터링된 대량의 테스트 케이스 목록 로딩 시 성능 향상을 위해 페이지네이션을 적용하였습니다.
- **동적 Jira 이슈 링크**: 활성화된 설정을 기반으로 Jira 이슈 링크가 동적으로 생성되고 연결되도록 개선하였습니다.
- **폴더 경로 표시 정교화**: 필터링된 테스트 케이스 다이얼로그에 누락된 폴더 경로 정보를 추가하여 컨텍스트 파악을 용이하게 하였습니다.
- **트리 선택 로직 최적화**: 테스트 케이스 트리의 체크박스 선택 시 폴더를 제외하고 실제 테스트 케이스만 정확히 선택되도록 로직을 수정하였습니다.
