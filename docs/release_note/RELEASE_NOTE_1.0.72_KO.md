# Release Note - v1.0.72

## [1.0.72] - 2026-04-24

안녕하세요! 이번 1.0.72 업데이트에서는 **AI를 활용한 자동화 기능**과 **RAG(검색 증강 생성) 시스템의 통합**을 한 단계 더 발전시켰습니다. 테스트 케이스 작성과 관리가 그 어느 때보다 똑똑하고 편리해졌습니다.

### 주요 변경 사항

#### 🤖 AI가 테스트 케이스 이름을 대신 지어드려요
- **이름 자동 생성**: 이제 AI가 테스트 케이스의 내용을 분석하여 가장 적합한 제목과 메타데이터를 자동으로 생성해 줍니다. 
- **수동/자동 모드 지원**: 사용자가 직접 버튼을 눌러 생성할 수도 있고, 필요한 경우 자동으로 제안받을 수도 있어 작업 효율이 획기적으로 높아졌습니다.

#### 🧠 RAG 시스템이 더 정교해졌어요
- **개별 벡터화 등록**: 특정 테스트 케이스를 RAG 시스템에 즉시 등록하여 AI가 학습하도록 할 수 있는 기능을 추가했습니다.
- **실시간 상태 확인**: 내가 등록한 테스트 케이스가 RAG 시스템에 잘 반영되었는지 실시간으로 UI에서 확인할 수 있습니다.
- **LLM 연동 최적화**: RAG 설정 상태에 따라 AI 모델(LLM)이 최적의 답변을 내놓을 수 있도록 내부 로직을 동기화했습니다.

#### 🏗️ 더 빠르고 튼튼한 시스템 구조 (Refactoring)
- **트리 컴포넌트 최적화**: 테스트 케이스 목록을 보여주는 트리 구조를 전면 리팩토링하여 대량의 데이터를 다룰 때도 훨씬 부드럽게 작동합니다.
- **모듈화 강화**: 복잡했던 코드를 기능별로 분리하여 앞으로 더 빠르고 안정적인 기능 업데이트가 가능해졌습니다.

#### 📝 가이드 및 문서 최신화
- **AI 에이전트 가이드 업데이트**: `GEMINI.md`를 새롭게 추가하고 `AGENTS.md`, `CLAUDE.md`를 개선하여 AI와 함께하는 개발 환경을 더 명확하게 정리했습니다.
