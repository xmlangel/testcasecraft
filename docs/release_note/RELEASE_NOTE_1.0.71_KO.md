# Release Note - v1.0.71

## [1.0.71] - 2026-04-23

안녕하세요! 이번 1.0.71 업데이트에서는 **테스트 관리의 안정성을 대폭 강화**하고, **사용자 경험을 방해하던 세밀한 버그들을 해결**하는 데 집중했습니다. 시스템이 더 견고해지고 사용이 매끄러워졌습니다.

### 주요 변경 사항

#### 🛠️ 테스트 케이스 관리 기능이 더 튼튼해졌어요
- **빈 트리에서도 문제없어요**: 테스트 케이스가 하나도 없는 빈 상태에서도 새로운 케이스를 막힘없이 추가할 수 있도록 수정했습니다.
- **대량 삭제도 안심하세요**: 수많은 테스트 케이스를 한꺼번에 삭제할 때 발생하던 시스템 충돌 문제를 해결하여, 이제 대규모 작업도 안정적으로 처리됩니다.
- **시작이 더 빨라졌어요**: 화면 로딩 시 발생하던 내부 초기화 오류를 잡아내어 서비스 이용이 더 쾌적해졌습니다.

#### 🔒 데이터 보호와 시스템 무결성이 좋아졌어요
- **완료된 결과는 안전하게**: 이미 '완료(COMPLETED)'된 테스트 결과가 실수로 수정되지 않도록 보호 로직을 추가했습니다.
- **똑똑한 에러 메시지**: 결과 수정이 불가능한 경우, 왜 안 되는지 한국어와 영어로 친절하게 설명해 줍니다.
- **데이터베이스 최적화**: 내부 데이터 구조(JPA)를 정돈하여 보이지 않는 곳의 성능과 안정성을 높였습니다.

#### 🎨 더 매끄럽고 편안한 화면을 제공해요
- **포커스가 자연스러워요**: 메뉴를 열거나 닫을 때 포커스가 튀던 현상을 수정하여 키보드와 마우스 조작이 더 부드러워졌습니다.
- **화면이 더 가벼워졌어요**: 중복된 코드들을 정리하고 접근성을 개선하여 전반적인 UI 반응 속도가 향상되었습니다.
