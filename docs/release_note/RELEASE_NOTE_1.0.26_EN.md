# Release Notes - v1.0.26

## 🚀 Key Changes

### 📈 UI & User Convenience
- **Pagination Improvements**: Enhanced pagination logic in test execution lists and automation results for more robust handling of empty data states.
- **Intuitive Status Display**: Refined color coding for test result states to improve visibility.

### 🛠 System Foundation
- **Dynamic OpenAPI Versioning**: Automated OpenAPI version settings using BuildProperties for Swagger UI and other integrations.
- **License & Documentation**: Officially designated Apache License 2.0 and updated Swagger API documentation contact and detail info.

### 🐛 Key Bug Fixes ([fix])
- **[fix] Standalone Project Visibility**: Fixed an issue where standalone projects were not displayed on the dashboard in certain scenarios.
- **[feat] Spring Boot DevTools**: Integrated DevTools for development efficiency and improved global error handling stack trace logic.
