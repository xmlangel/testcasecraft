# 릴리스 노트 - v1.0.28

## 🚀 주요 변경 사항

### 🔐 인증 및 보안
- **[feat] 세션 만료 대화상자 추가**: 사용자 세션 만료 시 갑작스러운 로그아웃 대신, 사유를 명시하는 다이얼로그를 추가하여 보안 안내를 강화했습니다.
- **인증 컨텍스트 업데이트**: 세션 관리 로직 최적화를 위해 Auth Context의 내부 상태 처리 방식을 개선했습니다.

### 🛠 리팩토링 및 안정화
- **로깅 표준화**: `console.log`를 전용 `debugLog`로 교체하여 운영 환경에서의 로그 노출을 제어하고 디버깅 효율을 높였습니다.
- **UI 반응성 개선**: 테스트 케이스 저장 시 전체 객체를 반환하도록 개선하고, 입력 폼의 헤더 UI를 정교하게 다듬었습니다.

### 🐛 버그 수정 ([fix])
- **테스트 케이스 트리 포커스 개선**: 테스트 케이스 삭제 후 트리의 포커스가 적절하게 이동하도록 로직을 보완하여 사용자 편의성을 높였습니다.
