# 릴리스 노트 - v1.0.33 (인프라 및 안정화)

## 🚀 주요 변경 사항

### 🧪 테스트 및 품질 보증

- **인증 API 테스트 자동화**: JSON 스키마 검증이 포함된 포괄적인 인증 API 테스트 세트를 구축하여 로그인 및 보안 로직의 안정성을 강화했습니다.
- **테스트 설정 마이그레이션**: 테스트 환경의 유연성을 위해 기존 `.properties` 기반 설정을 `.yml` 형식으로 통합 마이그레이션했습니다.

### ⚙️ 시스템 및 인프라

- **디렉토리 구조 표준화**: Docker 관련 설정 디렉토리를 통합하고 명칭을 직관적으로 변경하여 빌드 환경의 일관성을 높였습니다.
- **데이터 안정성**: 애플리케이션 초기화 시 발생할 수 있는 외래 키(Foreign Key) 제약 조건 위반 문제를 정밀하게 진단하고 수정했습니다.
