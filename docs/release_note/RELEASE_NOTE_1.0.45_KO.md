# Release Note - v1.0.45

## [1.0.45] - 2026-03-18

### 주요 변경 사항

#### 🚀 새로운 기능 (Features)
- **테스트 플랜 다중 선택 지원**: 이제 여러 개의 테스트 플랜을 동시에 선택하여 관리할 수 있습니다.
- **폴더 계층 구조 보기**: 테스트 케이스를 폴더 깊이(Depth)에 따른 계층 구조로 확인할 수 있는 기능을 추가했습니다.
- **URL 공유 및 새로고침 개선**: 현재 페이지의 상태가 URL에 반영되어, 새로고침 시에도 상태가 유지되고 링크 공유가 가능해졌습니다.
- **지라(JIRA) 이슈 URL 매핑**: 테스트 케이스와 지라 이슈 간의 연결을 위한 URL 매핑 기능을 추가했습니다.
- **비고(Notes) 클립보드 복사 기능**: 테스트 실행 결과의 비고 내용을 한 번의 클릭으로 클립보드에 복사할 수 있는 기능을 추가했습니다. (상세 페이지 및 테이블 뷰 지원)

#### 🛠 버그 수정 (Fixes)
- **프로젝트 필터 오류 수정**: 프로젝트별로 필터링 시 발생하던 기능 오류를 해결했습니다.
- **테스트 결과 리스트 상태 유지**: 테스트 결과 리스트에서 페이지 이동 시에도 이전 상태가 유지되도록 개선했습니다.

#### 📝 문서 업데이트 (Documentation)
- **사용자 매뉴얼 업데이트**: 새로운 기능(폴더 구조, 다중 테스트 플랜 등)에 대한 설명을 매뉴얼에 반영했습니다.
