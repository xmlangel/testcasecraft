# Release Note - v1.0.87

## [1.0.87] - 2026-06-12

안녕하세요! 1.0.87은 화면 가독성 중심의 업데이트입니다. 결과 입력 화면에 **모두 펼치기**와 **자동 줄바꿈(워드랩)** 보기 옵션이 추가되어 작은 화면에서도 케이스 본문을 편하게 볼 수 있고, 결과 상세 목록·대시보드의 표시 깨짐을 수리했습니다.

### 주요 변경 사항

#### 📱 결과 입력 화면 보기 옵션 (신규)

테스트 케이스 결과 입력 화면의 케이스 본문 상단에 보기 옵션 토글 2개가 추가되었습니다.

* **모두 펼치기**: 테스트 단계 셀의 상하 스크롤(높이 제한)을 해제하고 내용을 전부 표시합니다. 기본값은 기존과 동일한 스크롤 모드입니다.
* **줄바꿈(자동 워드랩)**: SQL 코드블록과 긴 문자열까지 모두 자동 줄바꿈되어, **모바일 등 작은 화면에서 가로 스크롤 없이** 케이스 본문 전체(설명·사전조건·단계·기대결과)를 읽을 수 있습니다.
* 선택한 옵션은 브라우저에 저장되어 **다음에 열 때도 유지**됩니다. 결과 입력 화면과 이전 결과 수정 다이얼로그가 같은 설정을 공유합니다.

#### 🛠️ 테스트 결과 상세 목록 표시 수리 (개선)

* **케이스명 겹침 해소**: 긴 테스트케이스 이름이 편집·상세보기 아이콘 밑으로 흘러 겹쳐 보이던 문제를 수정했습니다. 이름은 최대 2줄로 줄바꿈되고, 전체 이름은 호버 툴팁으로 확인합니다.
* **비고 툴팁 코드블록 가독성**: 어두운 툴팁 안에서 마크다운 코드블록이 흰 배경으로 렌더링되어 글자가 보이지 않던 문제를 수정했습니다 (비고·설명 컬럼 공통).

#### 📊 대시보드 빈 상태 표시 수정 (개선)

* 데이터가 없는 차트(결과 추이·열린 실행 결과·담당자별 결과)의 안내 문구가 모바일에서 한 글자씩 세로로 깨져 보이던 문제를 수정했습니다.

#### 🎨 스플래시 로고 통합 (정리)

* 초기 로딩 화면의 로고를 배경 투명 PNG 한 장으로 통합했습니다. 라이트/다크 테마별 JPG 2장과 로고 교체 스크립트를 제거하여 자산 관리가 단순해집니다.

#### ⚙️ 기타

* GitHub Actions를 Node.js 24 호환 버전으로 업데이트했습니다 (checkout v6, setup-java v5, docker login/buildx v4, gh-release v3).
* Docker Hub README에 영문 "How to Run"(.env / 쉘 변수 / --env-file) 섹션을 추가했습니다 (`docs/deployment/DOCKERHUB_README.md`).
