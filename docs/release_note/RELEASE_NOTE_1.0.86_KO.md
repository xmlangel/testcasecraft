# Release Note - v1.0.86

## [1.0.86] - 2026-06-10

안녕하세요! 1.0.86은 테스트 결과 리포팅을 강화한 업데이트입니다. 실행 단위로 **QA 총평**을 마크다운으로 기록하고 고급 내보내기 PDF에 함께 출력할 수 있으며, 이전 실행 결과의 노트를 **마크다운/텍스트 형식으로 전환**해 볼 수 있습니다.

### 주요 변경 사항

#### 📝 실행별 QA 총평 (신규)

테스트 실행을 마친 뒤 전체 소견을 기록하는 QA 총평 기능을 추가하였습니다.

* **작성 위치**: 테스트결과 화면의 **상세 테이블 보기** 탭에서 **테스트 실행 필터를 선택하면** 결과 테이블 위에 "QA 총평" 패널이 표시됩니다.
* **마크다운 입력**: **[총평 작성]**(또는 **[수정]**) 버튼으로 마크다운 에디터가 열리고, 입력 중 **라이브 프리뷰**로 결과를 바로 확인할 수 있습니다. 최대 10,000자.
* **실행 단위 저장**: 총평은 테스트 실행(execution) 단위로 저장되며, 패널 헤더에 **작성자와 마지막 수정 시각**이 표시됩니다.
* **고급 내보내기 연동**: PDF 리포트의 통계 요약 다음, **"상세 테스트 결과 리스트" 제목 바로 위**에 "💬 QA 총평"(작성자 포함)이 출력됩니다. 마크다운 서식은 일반 텍스트로 변환됩니다. Excel·CSV에는 포함되지 않습니다.

#### 📄 이전 실행 결과 — 노트 보기 형식 토글 (개선)

* 테스트 실행 목록의 **이전 실행 결과** 다이얼로그 우측 상단에 **마크다운 / 텍스트** 토글이 추가되었습니다.
* 마크다운 형식이 아닌 노트가 깨져 보이던 문제를 해결합니다 — 텍스트 모드는 노트를 **원문 그대로** 표시합니다.
* 선택한 보기 형식은 브라우저에 저장되어 **다음에 열 때도 유지**됩니다.

### 📖 사용자 매뉴얼

* 한국어/영문 매뉴얼에 위 변경 사항을 반영하였습니다 — §8-1 "이전 실행 결과 다이얼로그 — 노트 보기 형식 토글", §9-1 "QA 총평" 신규 섹션, §4-7 북마크 화면 캡처 추가.
* 헤더 변경으로 깨졌던 사용자 메뉴 캡처 3장을 수리하고, 신규 화면 캡처(북마크·이전 결과 다이얼로그·QA 총평)를 자동 캡처 대상에 추가하였습니다.

### 업그레이드 안내

* DB 컬럼(`qa_summary` 등)은 기동 시 자동으로 추가됩니다(별도 마이그레이션 불필요).
* **신규 UI 문구(번역 키)가 기존 서버 캐시에 가려질 수 있습니다.** 업그레이드 후 영어 화면에 한국어 라벨이 보이면 관리자 토큰으로 `POST /api/i18n/cache/clear`를 1회 호출하세요.
* 1.0.85의 즐겨찾기·북마크, 실행 목록 자동 새로고침은 [RELEASE_NOTE_1.0.85_KO.md](RELEASE_NOTE_1.0.85_KO.md)를 참고하세요.
