# Release Note - v1.0.50

## [1.0.50] - 2026-03-25

### 주요 변경 사항

#### 🐞 버그 수정 (Bug Fixes)

- **테스트 케이스 트리 UI 안정성 강화**:
  - 가상화 트리 도입 이후 발생하던 항목 선택(Selection) 오류를 수정하여 상세 폼과의 연동성을 복구하였습니다.
  - 폴더 또는 테스트 케이스를 트리 내에 추가할 때 레이아웃이 겹치거나 비정상적으로 렌더링되는 문제를 해결하였습니다.
  - 트리 컴포넌트 내의 런타임 코드 오류(정의되지 않은 핸들러 및 누락된 라이브러리 참조)를 전면 수정하였습니다.
- **JIRA 연동 상태 표시기 수정**: `JiraStatusIndicator` 컴포넌트에서 초기화되지 않은 변수를 참조하여 발생하던 `ReferenceError`를 신속하게 해결하였습니다.
- **MUI Tooltip 콘솔 경고 제거**: 비활성화된 버튼이나 비정상적인 자식 요소를 `Tooltip`과 함께 사용할 때 발생하던 `Material-UI`의 콘솔 경고 메시지를 레이아웃 래퍼(`<span>`) 추가를 통해 완벽히 차단하였습니다.
- **설정 저장 시 데이터 보호 강화**: 애플리케이션 설정을 업데이트할 때 기존에 저장되어 있던 API 키 값이 빈 값으로 덮어씌워져 삭제되는 치명적인 유실 문제를 방지하도록 저장 로직을 수정하였습니다.
