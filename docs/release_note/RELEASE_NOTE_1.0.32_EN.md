# Release Notes - v1.0.32

## 🚀 Key Changes

### 📊 UI/UX & Design

- **Sticky Spreadsheet Header**: Fixed the header (title) in the test case spreadsheet view while scrolling to improve readability when editing large datasets.
- **Enhanced Dark Mode Styling**: Refined dark mode styling for the version comparison detail view and key UI components for better visual consistency.

### 🏷 Visibility Improvements

- **Display ID Exposure**: Now displays the user-friendly `displayId` instead of internal system IDs in test case details, result headers, and test execution headers.
