# Release Note - v1.0.73

## [1.0.73] - 2026-04-24

안녕하세요! 이번 1.0.73 업데이트에서는 **RAG 기반 지능형 검색 기능**을 대폭 강화하고, **프로젝트 단위 데이터 격리 및 보안**을 강화했습니다. AI 채팅이 훨씬 더 정확하고 풍부한 답변을 드릴 수 있게 되었습니다.

### 주요 변경 사항

#### 🔍 RAG 지능형 검색 고도화
- **쿼리 의도 분석**: AI가 사용자의 질문 의도를 분석하여 단순 벡터 검색을 넘어, 실시간으로 데이터베이스에서 실제 테스트 케이스 정보를 조회해 더욱 정확한 답변을 제공합니다.
- **실시간 DB 데이터 연동**: RAG 채팅 서비스가 현재 프로젝트의 실제 데이터(테스트 케이스 통계, 목록 등)를 즉시 참조하여 최신 정보 기반의 응답을 생성합니다.
- **백엔드 템플릿 처리 이관**: 테스트 케이스 템플릿 주입 로직이 프론트엔드에서 백엔드(`RagQueryAnalyzer`)로 이관되어, 더 일관되고 안정적인 AI 응답 품질을 보장합니다.

#### 🔒 보안 강화: 프로젝트 데이터 격리
- **엄격한 프로젝트 범위 제한**: RAG 검색 및 SQL 실행 시 현재 사용자가 속한 프로젝트의 데이터만 접근할 수 있도록 격리 정책을 강화했습니다.
- **SQL 실행 보안 검증**: AI가 내부적으로 수행하는 SQL 쿼리에 대해 보안 유효성 검사를 추가하여 의도치 않은 데이터 접근을 원천 차단합니다.

#### 🧰 RAG 내부 로직 최적화 (Refactoring)
- **쿼리 분석기 개선**: `RagQueryAnalyzer`의 분석 로직을 고도화하여 사용자 의도를 더 정밀하게 파악하고 적절한 데이터를 선택적으로 조회합니다.
- **데이터 요약 로직 개선**: `RagDataSummarizer`가 조회한 데이터를 더 간결하고 AI 친화적인 형태로 요약하도록 개선되었습니다.
- **검색 API 정확도 향상**: FastAPI RAG 서비스의 벡터 유사도 검색 결과 정확도가 향상되었습니다.
