# Release Note - v1.0.52

## [1.0.52] - 2026-03-26

### 주요 변경 사항

#### 🚀 주요 기능 및 개선 사항 (Features & Enhancements)

- **대시보드 API 성능 대폭 최적화 (ICT-130)**:

  - 복잡한 데이터 집계를 효율적으로 처리하기 위해 CTE(Common Table Expressions) 및 Native SQL 도입.
  - 서비스 레이어에서의 N+1 쿼리 문제를 제거하고 벌크 집계(Bulk Aggregation) 방식을 적용하여 응답 시간을 **10초 이상에서 500ms 미만으로 약 95% 단축**했습니다.
  - 데이터 불일치 방지를 위해 DTO 매핑 및 null 안전 처리를 강화하여 JSON 스키마 검증 신뢰도를 높였습니다.

- **JIRA 연동 모듈 고도화 (ICT-JIRA)**:

  - **다중 JIRA ID 지원**: 하나의 테스트 결과에 여러 개의 JIRA 이슈를 동시에 연결하고 관리할 수 있도록 개선되었습니다.
  - **JIRA 코멘트 미리보기**: JIRA 코멘트 작성 시 마크다운 형식이 어떻게 렌더링되는지 제출 전 미리 확인할 수 있는 기능을 추가했습니다.
  - **이슈 상세 정보 자동 동기화**: JIRA ID 입력 시 해당 이슈의 상태, 요약, 우선순위 등을 실시간으로 페칭하여 UI에 표시합니다.
  - **워크플로우 개선**: 이슈 생성 후 즉시 해당 이슈로 이동하거나, 생성된 ID를 자동으로 입력 필드에 채워주는 편의 기능을 강화했습니다.

- **SBTM (Session-Based Test Management) 핵심 엔진 도입**:
  - 탐색적 테스팅(Exploratory Testing)을 지원하기 위한 세션 기반 테스트 관리 체계의 기반을 마련했습니다.
  - 테스트 차터(Charter) 관리 및 세션 라이프사이클(시작/일시중지/재개/종료/제출/승인) API와 상태 전이 로직을 구현했습니다.
  - 세션별 중단 시간 집계 및 승인 이력 관리 기능을 통해 테스트 생산성 지표 산출의 기반을 구축했습니다.

#### 🐞 버그 수정 (Bug Fixes)

- **JIRA 연동 이슈 해결**:
  - JIRA 이슈 생성 시 마크다운 형식이 올바르게 전달되지 않던 렌더링 문제를 수정했습니다.
  - 이슈 생성 다이얼로그의 내비게이션 버튼 및 아이콘 미표기 오류를 해결했습니다.
- **UI/UX 안정화**: 대시보드 차트 데이터가 프로젝트 변경 시 간헐적으로 초기화되지 않던 현상을 수정했습니다.
