# Release Note - v1.0.69

## [1.0.69] - 2026-04-16

이번 업데이트에서는 Google 스프레드시트와의 연동을 더욱 안전하게 만들고, 복잡했던 설정들을 한데 모아 사용하기 편리하도록 개선했습니다. 또한, 화면 곳곳에서 필요한 정보를 더 쉽게 확인할 수 있도록 UI를 다듬었습니다.

### 주요 변경 사항

#### 🚀 더 똑똑하고 안전해진 Google 스프레드시트 연동
- **데이터를 끝까지 안전하게**: 데이터를 저장하거나 업데이트할 때, 내부적으로 꼼꼼한 확인 과정을 거친 후 한 번에 처리하도록 개선했습니다. 이제 동기화 도중 데이터가 꼬이거나 유실될 걱정 없이 안심하고 사용하실 수 있습니다.
- **문제가 생기면 친절하게 안내**: 파일을 올릴 때 내용이 충돌하거나 오류가 생기면, 무엇이 문제인지 상세하게 알려드립니다. 이제 오류 메시지를 보고 바로 수정할 수 있습니다.
- **고유 번호(Display ID)를 어디서나 확인**: 테스트케이스의 고유 번호를 목록이나 화면 상단에서 바로 볼 수 있게 했습니다. 다른 문서와 대조할 때 훨씬 편리해집니다.

#### ✨ 더 간결하고 편리해진 화면(UI/UX)
- **복잡한 설정은 이제 안녕**: 따로 떨어져 있어 찾기 힘들었던 Google 스프레드시트 설정을 "내 프로필" 안으로 옮겼습니다. 이제 내 정보와 외부 연동 설정을 한곳에서 편하게 관리하세요.
- **이름 바꾸기가 더 쉬워졌습니다**: 테스트케이스 이름을 바꿀 때 편집 화면이 깨지거나 포커스를 놓치는 불편함을 없애기 위해 전용 대화창을 만들었습니다. 이제 더 깔끔하고 정확하게 이름을 바꿀 수 있습니다.
- **메뉴는 더 깔끔하게**: 자주 사용하지 않는 "JSON" 올리기 옵션을 화면에서 숨겨 메뉴를 더 보기 좋게 정리했습니다. (물론 기존에 쓰고 계신 기능은 내부적으로 그대로 유지됩니다.)

#### 🛠️ 시스템의 보이지 않는 곳까지 꼼꼼하게 최적화
- **내부 데이터 처리 방식 개선**: 시스템이 정보를 불러오는 방식을 더 튼튼하게 고쳤습니다. 이제 데이터가 갑자기 안 보이거나 오류가 나는 현상이 줄어듭니다.
- **개발 로그 정리**: 시스템 내부에서 기록하는 복잡한 로그들을 정리하여, 실제 확인이 필요한 정보만 깔끔하게 남겼습니다.
- **준비된 파일로 바로 시작**: 엑셀이나 CSV 파일을 처음 올릴 때 참고할 수 있는 샘플 파일들을 실제 시스템 형식에 딱 맞게 고쳤습니다. 이제 샘플 파일 그대로 내용을 채워 넣기만 하면 바로 사용할 수 있습니다.

#### 🔒 보안 및 기타 개선
- **로그인 유지 기능 보완**: 서버를 재시작하거나 네트워크 환경이 바뀌어도 로그인이 더 안정적으로 유지되도록 보완했습니다.
