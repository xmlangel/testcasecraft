# Release Notes - v1.0.33 (Stability & Infra)

## 🚀 Key Changes

### 🧪 Testing & QA
- **Auth API Test Automation**: Built a comprehensive authentication API test suite with JSON schema validation to ensure stable login and security logic.
- **Test Configuration Migration**: Migrated test settings from `.properties` to `.yml` for improved flexibility in test environments.

### ⚙️ System & Infrastructure
- **Directory Structure Standardization**: Unified and renamed Docker-related configuration directories for better consistency in build environments.
- **Data Integrity**: Diagnosed and resolved potential foreign key constraint violations during application data initialization.
