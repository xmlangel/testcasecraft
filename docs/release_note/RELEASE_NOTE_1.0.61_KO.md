# Release Note - v1.0.61

## [1.0.61] - 2026-04-03

### 주요 변경 사항

#### 🚀 JIRA 통합 기능 고도화 (JIRA Integration Enhancements)

- **JIRA 이슈 URL 파싱 및 자동 추출**: JIRA 이슈 URL을 직접 입력하거나 붙여넣을 때 이슈 키를 자동으로 추출하는 `JiraKeyUtils`를 도입하였습니다.
- **다중 이슈 지원 및 UI 개선**: 여러 개의 이슈 키를 동시에 조회하고 연결할 수 있도록 개선되었으며, 이슈의 상태 및 우선순위 파싱 로직을 강화하여 더 정확한 정보를 표시합니다.
- **검색 상태 동기화**: 링커 컴포넌트와 부모 컴포넌트 간의 검색 쿼리 상태를 동기화하여 사용자 경험을 개선하였습니다.

#### 📊 테스트 결과 대시보드 및 통계 개선 (Dashboard & Statistics)

- **실행 기반 비교 뷰 및 파이 차트**: 테스트 결과 대시보드에 실행(Execution) 단위의 데이터 시각화를 위한 파이 차트를 추가하여, 실행별 진행 상황을 직관적으로 비교할 수 있습니다.
- **통계 계산 로직 정확도 향상**: 기존 결과 맵 기반의 카운팅 방식에서 테스트 케이스 ID 목록을 직접 참조하는 방식으로 변경하여, 데이터 중복으로 인한 통계 수치 왜곡 문제를 해결하였습니다.
- **통계 계산 로직 중앙화**: 테스트 실행 요약 및 진행률 계산 로직을 `testResultConstants.js`로 통합하여 관리 효율성을 높였습니다.

#### ✨ 관리 도구 및 UI 개선 (Admin Tools & UI)

- **버전 히스토리 UI 개선**: 테스트 케이스 버전 히스토리 페이지의 로컬라이징(다국어 지원)을 강화하고, 상세 UI 레이아웃을 다듬어 가독성을 높였습니다.
- **첨부파일 다운로드 오류 처리 강화**: 첨부파일 다운로드 시 발생할 수 있는 예외 상황에 대한 핸들링과 로깅을 추가하여 안정성을 확보하였습니다.
