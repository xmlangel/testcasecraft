# Release Note - v1.0.74

## [1.0.74] - 2026-05-10

안녕하세요! 이번 1.0.74 업데이트에서는 **탐색 세션(Exploratory Session) 기능**을 대폭 강화하고, **Material 3 디자인 시스템**을 도입하여 UI/UX를 개선했습니다. 또한 사용자의 편의를 위해 UI 한글화를 진행했습니다.

### 주요 변경 사항

#### 🕵️‍♂️ 탐색 세션(Exploratory Session) 기능 고도화
- **승인 및 반려 워크플로우 추가**: Debrief 탭에 리드 리뷰 코멘트 기능과 함께 세션 승인/반려 워크플로우를 도입했습니다.
- **자동 저장 기능**: 세션 제출 전 자동 저장 기능을 구현하여 데이터 유실을 방지합니다.
- **UI 편의성 개선**: 세션 테스트 및 버그 섹션에 접기/펼치기(Collapsible) 기능을 추가하고, 첨부파일 설명을 업데이트할 수 있도록 개선했습니다.
- **필수 항목 강화**: 탐색 세션 저장 시 테스트 차터(Charter) 선택을 필수화하여 체계적인 테스트를 유도합니다.

#### 🎨 디자인 시스템 및 UI 개선
- **Material 3 도입**: Material 3 디자인 시스템 문서화, 컴포넌트 프리뷰, 토큰 정의를 완료하고 적용을 시작했습니다.
- **디자인 통일**: 탐색 세션의 디자인 시스템을 통일하고 하드코딩된 스타일을 제거했습니다.
- **한글화(Localization)**: 탐색 세션 UI 레이블과 상태 매핑을 한국어로 로컬라이즈하여 가독성을 높였습니다.

#### 🔐 시스템 및 기타 개선
- **로그인 페이지 접근성**: 로그인 페이지에 대한 공개 접근을 허용하고 라우트 리다이렉션을 개선했습니다.
- **코드 정리**: 날짜 포맷팅 로직을 단순화하는 등 코드 품질을 개선했습니다.
