# Release Note - v1.0.76

## [1.0.76] - 2026-05-22

안녕하세요! 이번 1.0.76 업데이트에서는 **회원가입 및 데이터 무결성 에러 메시지를 한국어로 친화화**하고, **TestCaseCraft API를 LLM 클라이언트(Claude Desktop·Cline·Cursor 등)에서 자연어로 호출할 수 있는 MCP 서버**를 새롭게 추가했습니다.

### 주요 변경 사항

#### 🤝 회원가입·에러 메시지 친화화

- **친화적인 회원가입 실패 응답**: 사용자명·이메일이 중복되면 `이미 사용 중인 사용자 이름입니다.` / `이미 등록된 이메일입니다.` 와 같이 명확한 한국어 메시지와 함께 `field`(중복 항목)를 반환하도록 변경했습니다.
- **HTTP 상태 코드 정규화**: 중복 가입은 `400 Bad Request` → `409 Conflict` 로 변경되어 클라이언트가 충돌 상황을 구분하기 쉬워졌습니다.
- **이메일 중복 사전 검증 추가**: 가입 요청 시 사용자명뿐 아니라 이메일도 사전 검사하여, 데이터베이스 예외 이전에 깔끔하게 차단합니다.
- **전역 데이터 무결성 예외 처리 개선**: PostgreSQL의 `duplicate key`, `foreign key`, `not-null` 위반 메시지를 사용자 친화 메시지로 변환하고, 컬럼명을 한국어 라벨(`이메일`·`사용자 이름`·`프로젝트 키` 등)로 매핑합니다. DB 내부 제약 이름(`uk_...`) 같은 민감 정보가 사용자에게 노출되지 않습니다.

#### 🤖 MCP 서버 신규 출시 (TestCaseCraft × LLM)

- **자연어로 TestCaseCraft 조작**: Claude Desktop, Cline, Cursor 등 MCP 호환 LLM 클라이언트에서 프로젝트·테스트케이스·실행·리포트·RAG 챗 등을 자연어로 조회·생성할 수 있습니다.
- **40여 개 도구 제공**: 인증(`auth_*`), 프로젝트, 테스트케이스(첨부 포함), 테스트 실행·결과·세션·플랜, 대시보드, Jira 연동, 조직·사용자, RAG, 시스템 헬스 등 433개 REST 엔드포인트를 도구 단위로 정리했습니다.
- **자동 인증·재로그인**: JWT 토큰을 안전하게 저장하고 만료 시 자동 갱신합니다.
- **첨부파일 업로드 지원**: 멀티파트 업로드로 테스트케이스에 로그·증빙 파일을 첨부할 수 있습니다.
- 자세한 설치·연동 방법은 `mcp-server/README.md` 를 참고하세요.

#### 🔧 MCP 도구 보강

- **조직 사용자 목록 도구 개선**: 관리자용 사용자 목록 엔드포인트로 전환하고, `role`(ADMIN/MANAGER/TESTER/USER)·`isActive` 필터를 추가했습니다.
