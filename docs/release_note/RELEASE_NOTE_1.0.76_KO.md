# Release Note - v1.0.76

## [1.0.76] - 2026-05-14

안녕하세요! 이번 1.0.76 업데이트에서는 새로운 디자인 시스템 도입과 AI 에이전트 기능 확장, 그리고 다국어 지원을 강화했습니다.

### 주요 변경 사항

#### 🎨 신규 디자인 시스템 (CreateSpace) 도입
- **새로운 스타일 시트 추가**: `createspace.css`를 통해 새로운 디자인 시스템을 적용했습니다.
- **UI 컴포넌트 개선**: `UserProfileDialog` 및 `ThemeContext`를 수정하여 사용자 인터페이스와 테마 전환 기능을 개선했습니다.

#### 🤖 AI 에이전트 및 스킬 확장
- **전문 에이전트 추가**: 디자인 시스템 QA, i18n 키 분류/작성, 프로필 UI 확장 등을 담당하는 특화된 Claude 에이전트들을 추가했습니다.
- **스킬 시스템 고도화**: 에이전트들이 사용할 수 있는 다양한 스킬(SKILL.md)을 추가하여 자동화 능력을 향상시켰습니다.

#### 🌐 다국어(i18n) 지원 강화
- **번역 데이터 업데이트**: 조직 및 사용자 관리 관련 다국어 키와 한/영 번역 데이터를 추가 및 수정했습니다.
