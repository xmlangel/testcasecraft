# 릴리스 노트 - v1.0.35

## 🚀 주요 변경 사항

### 📈 UI 혁신 및 가시성 개선 (JUnit 결과)

- **탭 네비게이션을 아코디언으로 교체**: JUnit 결과 화면의 기존 탭 기반 섹션을 확장/축소 가능한 아코디언(Accordion) 방식 프리셋으로 교체하여 제한된 공간에서 더 많은 정보를 효율적으로 확인할 수 있게 되었습니다.
- **통계 가독성 향상**: 아코디언 컴포넌트 내에 통계 대시보드를 배치하여 주요 테스트 결과 지표를 더 직관적으로 파악할 수 있도록 UI를 개선했습니다.
- **자동화된 상태 관리**: 아코디언의 접힘/펼침 상태가 유지되도록 기능을 추가하고 관련 다국어(i18n) 번역을 완비했습니다.

### ⚙️ 시스템 설정 및 프로젝트 구조

- **Docker 빌드 경로 현행화**: `docker-compose-dev-spring` 디렉토리명을 `docker-compose-build`로 통합 변경하고 빌드 스크립트의 관련 경로를 모두 업데이트했습니다.
- **애플리케이션 버전 일괄 갱신**: 프론트엔드, 백엔드 및 Docker 설정의 버전을 1.0.35로 상향 조정했습니다.

### 📚 문서화 및 가이드

- **에이전트 가이드 현행화**: `CLAUDE.md`, `GEMINI.md`, `QWEN.md` 등 AI 에이전트를 위한 지침 문서의 프로젝트 기술 스택 및 버전 정보를 최신 상태로 동기화했습니다.
- **워크플로우 정립**: 릴리스 관리 및 버전 동기화에 관한 신규 워크플로우 지침을 추가했습니다.
