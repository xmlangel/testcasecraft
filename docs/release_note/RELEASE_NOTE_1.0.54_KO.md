# Release Note - v1.0.54

## [1.0.54] - 2026-03-27

### 주요 변경 사항

#### 🚀 주요 기능 및 개선 사항 (Features & Enhancements)

- **스프레드시트 입력 모드 강화**:
  - **폴더 기반 필터링 지원**: 이제 전체 테스트 케이스가 아닌 선택한 특정 폴더의 항목들만 스프레드시트 형태로 편집할 수 있어, 대규모 프로젝트에서의 작업 효율이 대폭 향상되었습니다.
  - **유효성 검증 로직 최적화**: 필터링된 보기 상태에서도 전체 프로젝트 구조를 참조하도록 개선되어, 상위 폴더 경로 확인 및 데이터 정합성 검증이 더욱 정확해졌습니다.

#### 🐞 버그 수정 (Bug Fixes)

- **스프레드시트 모드 오류 해결**:
  - **행 추가 기능 복구**: 컴포넌트 내부 함수 초기화 순서 문제로 인해 발생하던 `ReferenceError`를 수정하여 '행 추가' 및 '위/아래 행 삽입' 기능이 정상 작동하도록 복구했습니다.
  - **순서(Display Order) 보존**: 대량 저장 시 필터링 조건에 따라 테스트 케이스의 정렬 순서가 1로 초기화되던 문제를 해결하여 기존 정렬 상태가 안정적으로 유지되도록 개선했습니다.
