# Release Note - v1.0.70

## [1.0.70] - 2026-04-17

Hello! In version 1.0.70, we've focused on **making the system more stable**, **refining language support (Korean/English)**, and **making the UI more intuitive and beautiful** for a better user experience.

### Key Changes

#### ✨ More Natural Language Support
- **Natural Date Formatting**: Fixed awkward date displays in dashboards and lists. Dates now follow your language settings naturally.
- **Filled Translation Gaps**: Improved localized text for renaming test cases and profile views in both English and Korean.
- **English Help Guides**: Added an English version of the Google Sheets setup guide to assist our global users.

#### 🎨 Beautiful and Convenient UI
- **Refreshed Google Config Screen**: Redesigned the Google Sheets integration page with a modern, organized layout.
- **Quick Navigation**: Added direct navigation buttons to profile tabs (like Google Settings) for faster access.
- **Case IDs at a Glance**: Displayed unique Case IDs directly on the screen to improve work efficiency.

#### 🚀 Smarter and More Stable System
- **Reduced Error Messages**: Cleaned up technical error messages when configurations are missing, providing a cleaner console experience.
- **Smarter Google Sheets Integration**: Added URL validation and a "Remember Last Settings" feature for a smoother import/export flow.
- **Smoother Performance**: Optimized internal code to ensure smooth transitions and interactions without lag.

#### 🛠️ Bug Fixes & Others
- Fixed text overlapping and visibility issues in various screens.
- Resolved several minor bugs that could cause unexpected pauses or errors.
- Polished back-end logic to ensure your precious data is handled even more securely.
