# Release Note - v1.0.49

## [1.0.49] - 2026-03-24

### 주요 변경 사항

#### 🚀 주요 기능 (Features)
- **무한 스크롤(Infinity Scroll) 도입**: 테스트 실행 목록에 무한 스크롤 방식을 적용하여 기존의 페이지네이션보다 자연스럽고 끊김 없는 탐색 경험을 제공합니다.
- **테스트 실행 제목 링크**: 테스트 실행 상세 페이지의 제목을 클릭하면 즉시 목록으로 돌아갈 수 있는 바로가기 링크를 추가하여 탐색을 용이하게 하였습니다.

#### 🛠 개선 및 최적화 (Improvements & Optimizations)
- **테스트 케이스 트리 성능 획기적 개선**: 트리 렌더링 로직을 $O(N^2)$에서 $O(N)$으로 최적화하고 가상화(Virtualization)를 도입하여, 수만 개의 테스트 케이스가 포함된 대규모 프로젝트에서도 지연 없는 사용자 경험을 제공합니다.
- **테스트 실행 API 고도화**: N+1 쿼리 문제를 해결하고 서버 사이드 통계 계산 및 페이지네이션을 적용하여, 테스트 실행 목록 조회 속도를 초 단위에서 밀리초(ms) 단위로 대폭 단축하였습니다.
- **테스트 실행 진행률 계산 로직 정확도 향상**: 각 테스트 케이스의 최신 결과만 집계하도록 수정하여, 진행률이 100%를 초과하는 현상을 방지하고 데이터 신뢰성을 보장합니다.

#### 🐞 버그 수정 (Bug Fixes)
- **통계 필터 패널 오류 수정**: 페이징 처리된 데이터가 리스트 형태로 처리되지 않아 발생하던 `TypeError`를 해결하고 방어적인 데이터 검증 로직을 추가하였습니다.
- **UI 레이아웃 품질 개선**: 테스트 결과 입력 시 메모 필드가 전체 화면 모드일 때 부유형 메뉴(Floating Menu)가 가려지는 문제를 레이어(Z-index) 조정을 통해 해결하였습니다.
- **BERT 판정 스크립트 도구 수정**: `bert_llm_judge-file.py` 실행 시 특정 컬럼(`cand`) 누락으로 인해 발생하던 오류를 수정하여 모델 평가 도구의 안정성을 확보하였습니다.
