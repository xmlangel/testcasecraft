# Release Note - v1.0.75

## [1.0.75] - 2026-05-11

안녕하세요! 이번 1.0.75 업데이트에서는 시스템 안정성을 강화하고 테스트 실패 문제를 해결했습니다.

### 주요 변경 사항

#### 🛠 시스템 안정성 및 성능 개선
- **트랜잭션 지원 추가**: 데이터 정합성을 보장하기 위해 트랜잭션 처리를 강화했습니다.
- **성능 테스트 보안 강화**: 성능 테스트 실행 시의 보안성을 보완했습니다.
- **널 체크(Null Check) 추가**: 트리 렌더링 및 필터링 로직에 null 체크를 추가하여 데이터 업데이트 중 발생할 수 있는 런타임 에러를 방지했습니다.

#### 🐛 버그 수정
- **통합 테스트 인증 문제 해결**: 통합 테스트 환경에서 발생하던 인증 관련 오류를 해결했습니다.
