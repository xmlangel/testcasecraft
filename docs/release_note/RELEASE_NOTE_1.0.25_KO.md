# 릴리스 노트 - v1.0.25

## 🚀 주요 변경 사항

### 👨‍💻 관리자 기능 강화
- **[feat] 자동 Admin 계정 생성**: 시스템 최초 기동 시 관리자 계정이 없을 경우, 기본 Admin 계정을 자동으로 생성하도록 개선하여 초기 설정 편의성을 높였습니다.
- **[fix] RAG 문서 검색 안정화**: 특정 조건에서 RAG 문서 검색 시 발생하던 경로 탐색 오류를 디버깅하고 수정했습니다.
- **[fix] 로그 정리 및 i18n 보완**: 불필요한 콘솔 로그를 정리하고 누락되었던 다국어 번역을 전반적으로 보완했습니다.

### 📊 테스트 케이스 필드 대폭 확장
- **상세 메타데이터 추가**: 테스트 케이스에 사후 조건(Post-condition), 우선순위(Priority), 실행 유형(Execution Type), 테스트 기법(Test Technique), 태그(Tags) 필드를 새롭게 추가하여 데이터 구조를 체계화했습니다.

### 💾 설정 및 입력 편의성
- **입력 모드 상태 유지**: `localStorage`를 활용하여 사용자의 이전 입력 모드 상태를 저장함으로써 리프레시 후에도 작업 환경이 유지되도록 개선했습니다.
- **한글 입력 최적화**: 테스트 케이스 제목이나 설명 작성 시 한글 입력이 씹히지 않고 자연스럽게 처리되도록 UX를 보완했습니다.
