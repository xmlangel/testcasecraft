# 릴리즈 노트 - v1.0.44

## 🚀 주요 변경사항

### 🔗 JIRA 연동 및 상태 표시 개선

- **JIRA 이슈 상태 메시지 한글화**: "Unknown", "ERROR" 등 영문으로 표시되던 JIRA 이슈 상태를 "상태 확인 불가", "연동 오류" 등 친숙한 한글 메시지로 개선했습니다.
- **이슈 URL 매핑 강화**: JIRA 이슈 번호 클릭 시 해당 이슈 페이지로 정확하게 이동할 수 있도록 URL 매핑 로직을 보완했습니다.
- **Jira 스마트 리다이렉트(Smart Redirect)**: Jira 이슈 키(Key)를 기반으로 해당 이슈와 연관된 테스트 실행(Test Execution)을 자동으로 조회하고, 결과 입력 폼으로 즉시 이동하며 이슈 키를 자동 매핑하는 스마트 리다이렉트 기능을 도입했습니다.

### 🧩 사용자 경험(UX) 및 UI 최적화

- **테스트 실행 리스트 위치 복구**: 테스트 결과 상세 페이지에서 목록으로 되돌아올 때, 이전에 보던 스크롤 위치를 자동으로 유지하여 탐색 편의성을 높였습니다.
- **플로팅 메뉴(Floating Menu) 도입**: 테스트 결과 입력 및 상세 화면에서 주요 액션(결과 저장, JIRA 확인 등)을 언제든 수행할 수 있도록 하단 플로팅 메뉴를 추가했습니다.
- **레이아웃 반응형 개선**: 화면 크기에 따라 패딩 및 높이 등이 최적화되도록 UI 레이아웃을 리팩토링했습니다.

### 📊 데이터 집계 및 관리 기능 강화

- **테스트 통계 로직 개선**: 테스트 결과 통계 데이터 계산 시, 누적값이 아닌 최종 실행 결과를 기준으로 계산하도록 수정하여 정확도를 높였습니다.
- **관리자 메일 설정 기능**: 시스템 관리자가 서비스 내에서 직접 메일 서버(SMTP) 설정을 관리할 수 있는 인터페이스를 추가했습니다.
- **날짜 및 시간 표시 개선**: `date-fns` 라이브러리를 도입하여 날짜 포맷을 정교화하고, 테스트 실행 결과에 UTC 타임존 표시를 추가했습니다.

### 🛠️ 기타 개선 사항

- 프론트엔드 코드 내 날짜 처리 유틸리티 연동 오류 수정
- 전반적인 API 성능 모니터링 및 로깅 강화
