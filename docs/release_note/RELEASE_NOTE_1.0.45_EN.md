# Release Note - v1.0.45

## [1.0.45] - 2026-03-18

### Major Changes

#### 🚀 Features

- **Multiple Test Plan Support**: Added ability to select and manage multiple test plans simultaneously.
- **Folder Hierarchy View**: Introduced a hierarchical view for test cases based on folder depth.
- **Improved URL Sharing & Refresh**: Page states are now reflected in the URL, allowing for state persistence on refresh and easy link sharing.
- **JIRA Issue URL Mapping**: Added functionality to map Jira issue URLs to test cases for better integration.
- **Notes Clipboard Copy**: Added a button to copy the 'Notes' content from test execution results to the clipboard with a single click. (Supported in both detailed and table views)

#### 🛠 Fixes

- **Project Filter Bug Fix**: Resolved an issue where filtering by project was not functioning correctly.
- **Test Result List State Persistence**: Improved state management to maintain the list view state when navigating through test results.

#### 📝 Documentation

- **Manual Update**: Updated the user manual to include instructions for new features such as folder hierarchy and multiple test plans.
