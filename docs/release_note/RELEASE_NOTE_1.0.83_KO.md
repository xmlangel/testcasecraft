# Release Note - v1.0.83

## [1.0.83] - 2026-06-06

안녕하세요! 1.0.83은 1.0.82의 테스트케이스 화면 개편을 다듬는 후속 업데이트입니다. 중첩 폴더 구조에서의 케이스 열람을 개선하고, 도커 배포에서 인앱 매뉴얼이 열리지 않던 문제를 수정하였습니다.

### 주요 변경 사항

#### 📋 폴더 케이스 목록 — 하위 폴더 케이스 포함 (개선)

* `폴더 > 폴더 > 케이스`처럼 중첩된 구조에서 **상위 폴더만 선택해도 그 아래 모든 테스트케이스가 함께 표시**됩니다.
  - 기존에는 직속 항목까지만 보여서 하위 폴더 안의 케이스를 보려면 폴더를 한 단계씩 들어가야 했습니다.
  - 목록 구성: 직속 하위 폴더(탐색용) → 하위 폴더를 포함한 모든 케이스(폴더 단위 묶음, 표시 순서 유지).
* 하위 폴더 소속 케이스에는 **폴더** 컬럼이 자동으로 추가되어 선택한 폴더 기준의 상대 경로를 보여주고, 경로를 클릭하면 해당 폴더로 바로 이동합니다.
* 중첩 케이스가 없는 폴더는 기존과 동일한 단순 목록으로 표시됩니다.

#### 🐛 버그 수정

* 도커 배포에서 **인앱 매뉴얼/가이드 뷰어가 404로 열리지 않던 문제**를 수정하였습니다.
  - 원인: 매뉴얼·가이드 문서를 서버 작업 디렉토리에서만 읽어, 컨테이너(jar만 존재)에서는 문서를 찾지 못했습니다.
  - 수정: 매뉴얼(한/영)·스크린샷·가이드 문서를 배포 산출물(jar)에 동봉하고, 개발 환경에서는 기존처럼 저장소 문서를 직접 읽도록 폴백 구조를 적용하였습니다.

#### 📖 매뉴얼

* 사용자 매뉴얼(한/영) §4-1 "폴더 케이스 목록"에 하위 폴더 케이스 포함 동작을 반영하였습니다.

### 업그레이드 안내

* 별도 마이그레이션 없이 이미지 교체만으로 적용됩니다.
* 1.0.82의 변경 사항(폴더 전용 트리, 케이스 목록, 탭 배지 등)은 [RELEASE_NOTE_1.0.82_KO.md](RELEASE_NOTE_1.0.82_KO.md)를 참고하세요.
