# Release Note - v1.0.58

## [1.0.58] - 2026-03-31

### 주요 변경 사항

#### 🎨 UI/UX 개선 (UI/UX Enhancements)

- **테스트 결과 헤더 리팩토링**: 상단 스티키 레이아웃을 적용하여 스크롤 시에도 주요 정보가 유지되도록 개선하였으며, 상태별 통계 수치 시각화를 강화하고 상세 내비게이션 컨트롤을 추가하여 사용성을 높였습니다.
- **플로팅 메뉴 고도화**: 테스트 결과 상세 페이지의 플로팅 메뉴 UI를 더욱 현대적으로 개선하고, 테스트 편집 양식으로의 실행 컨텍스트 전달 로직을 보완하여 끊김 없는 작업 흐름을 지원합니다.
- **실행 히스토리 스타일 업데이트**: 히스토리 테이블의 가독성을 향상시키고 전반적인 컴포넌트 디자인을 시스템 테마(다크 모드 포함)와 일관되게 정렬하였습니다.

#### ✨ 신규 기능 (New Features)

- **테스트케이스 실행 히스토리 연결**: 각 테스트케이스 상세 조회 페이지 내에 '실행 히스토리' 탭을 추가하여, 해당 테스트케이스의 과거 수행 기록을 한눈에 확인할 수 있게 되었습니다.
- **실행 결과 상세 바로가기**: 히스토리 목록에서 특정 실행 건을 클릭하면 해당 테스트 결과 페이지로 즉시 이동할 수 있는 내비게이션 기능을 구현하였습니다.

#### 🛠️ 기능 개선 및 최적화 (Optimizations)

- **인증 강화 및 API 클라이언트는 통합**: `TestCaseExecutionHistory` 컴포넌트에서 사용하던 기존의 브라우저 기본 fetch 로직을 인증 정보가 포함된 공통 API 클라이언트로 교체하여 데이터 요청의 보안과 안정성을 강화하였습니다.
