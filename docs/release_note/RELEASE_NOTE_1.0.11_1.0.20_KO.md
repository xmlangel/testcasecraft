# 릴리스 노트 - v1.0.11 ~ v1.0.20 (기술 스택 전환 및 고도화)

## 🚀 주요 변경 사항

### 🎨 UI 프레임워크 현대화 (MUI Grid v2 마이그레이션)

- **MUI 최신 스택 적용**: MUI Grid v2로 전면 전환하고 deprecated된 prop들을 정리하여 UI 안정성과 성능을 확보했습니다. (MUI v5 → v7 업그레이드 포함)
- **TreeView 업그레이드**: MUI의 기존 TreeView를 새로운 `SimpleTreeView`로 마이그레이션하여 모던한 계층 구조 관리 인터페이스를 구축했습니다.

### 🤖 LLM 및 RAG 시스템 구축

- **RAG 관리 패널 도입**: LLM 설정 및 전역 문서 관리를 위한 어드민 패널을 추가했습니다.
- **템플릿 관리 API**: 사용자별/프로젝트별 LLM 템플릿을 관리하기 위한 전용 API 엔드포인트를 구현했습니다.
- **비디오 및 문서 프리뷰**: RAG 문서 검색 결과에서 문서와 비디오를 직접 확인할 수 있는 프리뷰 기능을 강화했습니다.

### 🔌 외부 연동 및 통합 (Jira)

- **Jira 상태 동기화**: 프로젝트 내 테스트 케이스와 Jira 이슈 간의 상태 동기화 기능을 강화했습니다.
- **다국어 지원**: Jira 코멘트 및 연동 관련 UI에 대한 다국어 번역을 보강했습니다.

### 🛡 시스템 보안 및 안정성

- **종속성 보안 패치**: Spring Boot 버전을 최신화하고, Bouncy Castle, Guava, Apache POI 등 주요 라이브러리의 보안 취약점을 해결했습니다.
- **토큰 자동 갱신**: JWT 토큰의 자동 갱신(Refresh Token) 기능을 도입하여 끊김 없는 사용자 세션을 제공합니다.
- **낙관적 잠금 (Optimistic Locking)**: 테스트 케이스 동시 수정 시 데이터 정합성을 보장하기 위한 낙관적 잠금 메커니즘을 적용했습니다.

### 🧪 테스트 실행성 강화

- **필터링 및 정렬**: 테스트 실행 목록에 상세 필터를 추가하고 데이터 로딩 속도를 개선했습니다.
- **이전 결과 수정**: 관리자 또는 본인이 과거의 테스트 결과를 수정하거나 삭제할 수 있는 기능을 추가했습니다.
