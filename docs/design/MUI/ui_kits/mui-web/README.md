# MUI Web UI Kit

Interactive recreation of Material UI component patterns as seen in the Figma community file.

## Screens
- **Job Directory** — Main landing view with sidebar filters, job cards, pagination
- **Log In** — Card-based authentication form
- **User Management** — Admin data table

## Usage
Open `index.html` to explore the interactive prototype. Click navigation items and buttons to move between screens.

## Components
- `AppBar.jsx` — Top application bar with nav icons
- `Button.jsx` — Contained / outlined / text variants
- `TextField.jsx` — Outlined text inputs with labels
- `Card.jsx` — Content cards with header / content / actions
- `Chip.jsx` — Filter chips with optional icon
- `Badge.jsx` — Notification badges
- `Dialog.jsx` — Modal dialogs
- `Sidebar.jsx` — Drawer-style filter sidebar
- `DataTable.jsx` — Sortable table with status chips
