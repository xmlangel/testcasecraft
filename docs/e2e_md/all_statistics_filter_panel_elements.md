# 통계 필터 패널 UI 요소

이 문서는 `src/main/frontend/src/components/StatisticsFilterPanel.jsx` 파일을 기반으로 통계 필터 패널 컴포넌트에 존재하는 주요 UI 요소를 식별하고 기록합니다.

## 1. 메인 카드 구조

- **`Card` 컴포넌트**
- **`CardContent`**

## 2. 헤더 영역

- **필터 아이콘 (`FilterList`)**
- **제목 (`Typography` h6)**: `통계 필터`
- **적용된 필터 개수 칩 (`Chip`)**: `X개 적용`
- **액션 버튼들**
  - **새로고침 버튼 (`Button` with `Refresh`)**: `새로고침`
  - **초기화 버튼 (`Button` with `Clear`)**: `초기화`

## 3. 필터 옵션

- **테스트 플랜 선택 (`FormControl` 내 `Select`)**
  - 라벨: `테스트 플랜`
  - 옵션: `전체 플랜`, 각 테스트 플랜 이름
- **테스트 실행 선택 (`FormControl` 내 `Select`)**
  - 라벨: `테스트 실행`
  - 옵션: `전체 실행`, 각 테스트 실행 이름
- **기간 선택 (`FormControl` 내 `Select`)**
  - 라벨: `기간`
  - 아이콘: `DateRange`
  - 옵션: `전체 기간`, `오늘`, `최근 1주`, `최근 1개월`, `최근 3개월`
- **보기 형태 선택 (`FormControl` 내 `Select`)**
  - 라벨: `보기 형태`
  - 옵션: `전체 개요`, `플랜별 비교`, `실행자별 비교`

## 4. 적용된 필터 표시

- **구분선 (`Divider`)**
- **텍스트 (`Typography`)**: `적용 중인 필터:`
- **각 필터 칩 (`Chip`)**: `플랜: {플랜이름}`, `실행: {실행이름}`, `기간: {기간}` (삭제 버튼 포함)
